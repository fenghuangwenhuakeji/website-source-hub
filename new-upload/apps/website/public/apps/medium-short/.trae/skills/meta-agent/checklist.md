# Meta-Agent 质量检查清单

## 使用说明

本文档用于确保Meta-Agent系统的质量和正确性。在执行Meta-Agent相关任务时，请逐项检查以下清单。

**检查级别说明：**
- **[P0]** - 关键项，必须100%通过
- **[P1]** - 重要项，建议通过
- **[P2]** - 优化项，可选通过

---

## 检查流程

### 开发阶段检查
1. 每完成一个功能模块，执行对应模块的检查
2. 修复所有P0级别问题
3. 记录P1/P2级别问题，安排后续优化

### 代码审查检查
1. 审查者使用此清单进行审查
2. 重点检查P0级别项
3. 提出改进建议

### 发布前检查
1. 完整运行所有检查项
2. 确保所有P0级别项通过
3. 修复所有阻塞性问题

### 定期回顾检查
1. 每月回顾检查清单
2. 根据实践经验更新清单
3. 优化检查流程

## 自动化检查脚本

```bash
#!/bin/bash
# meta_agent_checks.sh

echo "========================================"
echo "Meta-Agent Quality Checks"
echo "========================================"

# 代码格式检查
echo "[1/10] Checking code formatting..."
cargo fmt -- --check
if [ $? -ne 0 ]; then
    echo "❌ Code formatting check failed"
    exit 1
fi
echo "✅ Code formatting check passed"

# Clippy检查
echo "[2/10] Running Clippy..."
cargo clippy --all-targets --all-features -- -D warnings
if [ $? -ne 0 ]; then
    echo "❌ Clippy check failed"
    exit 1
fi
echo "✅ Clippy check passed"

# 编译检查
echo "[3/10] Checking compilation..."
cargo check --all-targets --all-features
if [ $? -ne 0 ]; then
    echo "❌ Compilation check failed"
    exit 1
fi
echo "✅ Compilation check passed"

# 单元测试
echo "[4/10] Running unit tests..."
cargo test --lib
if [ $? -ne 0 ]; then
    echo "❌ Unit tests failed"
    exit 1
fi
echo "✅ Unit tests passed"

# 集成测试
echo "[5/10] Running integration tests..."
cargo test --test '*'
if [ $? -ne 0 ]; then
    echo "❌ Integration tests failed"
    exit 1
fi
echo "✅ Integration tests passed"

# 文档测试
echo "[6/10] Running doc tests..."
cargo test --doc
if [ $? -ne 0 ]; then
    echo "❌ Doc tests failed"
    exit 1
fi
echo "✅ Doc tests passed"

# 测试覆盖率
echo "[7/10] Checking test coverage..."
cargo tarpaulin --out Stdout
if [ $? -ne 0 ]; then
    echo "❌ Coverage check failed"
    exit 1
fi
echo "✅ Coverage check passed"

# 文档生成
echo "[8/10] Building documentation..."
cargo doc --no-deps
if [ $? -ne 0 ]; then
    echo "❌ Documentation build failed"
    exit 1
fi
echo "✅ Documentation build passed"

# 安全审计
echo "[9/10] Running security audit..."
cargo audit
if [ $? -ne 0 ]; then
    echo "❌ Security audit failed"
    exit 1
fi
echo "✅ Security audit passed"

# 性能测试
echo "[10/10] Running performance tests..."
cargo test --test performance
if [ $? -ne 0 ]; then
    echo "❌ Performance tests failed"
    exit 1
fi
echo "✅ Performance tests passed"

echo "========================================"
echo "✅ All checks passed!"
echo "========================================"
```
