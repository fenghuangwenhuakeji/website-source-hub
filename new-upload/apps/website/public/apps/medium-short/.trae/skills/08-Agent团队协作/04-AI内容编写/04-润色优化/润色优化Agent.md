# 润色优化 Agent 配置

## 概述

润色优化Agent专注于内容的质量提升，包括语言润色、错误修正和SEO优化。

## Agent角色

### 1. 内容审核Agent
```yaml
name: "内容审核Agent"
description: "审核内容合规性"
responsibilities:
  - 合规检查
  - 价值观审核
  - 敏感词检测
  - 版权问题
skills:
  - 法规知识
  - 内容标准
  - 风险评估
```

### 2. 润色校对Agent
```yaml
name: "润色校对Agent"
description: "润色语言和校对错误"
responsibilities:
  - 语法检查
  - 错别字纠正
  - 语言润色
  - 风格统一
skills:
  - 语言规范
  - 写作技巧
  - 细致入微
```

### 3. SEO优化Agent
```yaml
name: "SEO优化Agent"
description: "优化搜索引擎排名"
responsibilities:
  - 关键词优化
  - 标题优化
  - 元数据优化
  - 内链外链
skills:
  - SEO知识
  - 关键词研究
  - 排名优化
```

## 协作流程

```
1. 内容审核Agent检查合规性
   ↓
2. 润色校对Agent优化语言
   ↓
3. SEO优化Agent提升搜索排名
   ↓
4. 最终质量检查
   ↓
5. 发布准备完成
```
