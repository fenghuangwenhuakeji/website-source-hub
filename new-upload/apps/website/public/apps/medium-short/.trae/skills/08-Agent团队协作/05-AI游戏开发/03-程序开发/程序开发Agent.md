# 程序开发 Agent 配置

## 概述

程序开发Agent专注于游戏逻辑实现和技术开发。

## Agent角色

### 1. 游戏逻辑Agent
```yaml
name: "游戏逻辑Agent"
description: "实现游戏核心逻辑"
responsibilities:
  - 玩法实现
  - 状态机
  - 事件系统
  - AI行为
skills:
  - C#/C++
  - Unity/Unreal
  - 设计模式
```

### 2. 引擎开发Agent
```yaml
name: "引擎开发Agent"
description: "开发和优化游戏引擎"
responsibilities:
  - 渲染优化
  - 物理系统
  - 性能优化
  - 工具开发
skills:
  - 图形学
  - 引擎架构
  - 优化
```

### 3. 工具开发Agent
```yaml
name: "工具开发Agent"
description: "开发游戏工具"
responsibilities:
  - 编辑器
  - 自动化工具
  - 资源管线
  - 调试工具
skills:
  - 工具开发
  - 编辑器扩展
  - Python
```

## 协作流程

```
1. 游戏逻辑Agent实现玩法
   ↓
2. 引擎开发Agent优化性能
   ↓
3. 工具开发Agent提供工具
   ↓
4. 集成测试
```
