## 1. 概述

### 1.1 项目背景

小说创作需要系统化的细纲规划。Outline Agent 运用"三界炼心法"进行角色塑造，结合情绪链条和爽点设计，生成完整的20章细纲。

### 1.2 目标范围

- 生成20章完整细纲
- 应用三界炼心法塑造角色
- 设计情绪链条
- 布局爽点设计
- 规划情节节奏

### 1.3 核心理念

**以角色塑造为起点，以细纲规划为手段，最终创造出让读者共情、追更、无法忘怀的"活"的人物。**

```
故事梗概 → 三界定锚 → 细纲生成 → 情绪设计 → 爽点布局 → 输出
```

## 3. 三界炼心法体系

### 3.1 命运界 (Bazi)

**四柱宫位**:
- 年柱: 祖上、出身、童年(1-18岁)
- 月柱: 父母、兄弟、青年(18-35岁)
- 日柱: 自己、配偶、中年(35-55岁)
- 时柱: 子女、晚年、归宿(55岁+)

**十神系统**:
- 比肩: 自我、独立
- 劫财: 竞争、欲望
- 食神: 才华、享受
- 伤官: 叛逆、创造力
- 正财: 稳定、务实
- 偏财: 投机、人际
- 正官: 规则、责任
- 七杀: 权力、斗争
- 正印: 庇护、知识
- 偏印: 孤独、智慧

### 3.2 原型界 (Tarot)

**大阿尔卡那** (22个核心原型):
- 0.愚人 → 21.世界
- 正逆位双重含义

**小阿尔卡那** (四大元素):
- 权杖: 意志、行动、创造力
- 圣杯: 情感、关系、直觉
- 宝剑: 思想、智慧、挑战
- 星币: 物质、工作、金钱

### 3.3 认知界 (MBTI)

**八大认知功能**:
- Ti: 内向思考 (学者)
- Te: 外向思考 (将军)
- Fi: 内向情感 (艺术家)
- Fe: 外向情感 (外交官)
- Si: 内向感觉 (历史学家)
- Se: 外向感觉 (运动员)
- Ni: 内向直觉 (先知)
- Ne: 外向直觉 (发明家)

## 4. 细纲结构

### 4.1 20章框架

| 阶段 | 章节 | 功能 | 字数占比 |
|------|------|------|---------|
| 开篇定调 | 1-3章 | 人物登场、世界建立 | 15% |
| 发展推进 | 4-7章 | 矛盾升级、能力提升 | 25% |
| 高潮酝酿 | 8-12章 | 重大转折、危机逼近 | 30% |
| 高潮爆发 | 13-16章 | 终极对决、情感顶点 | 20% |
| 收尾升华 | 17-20章 | 问题解决、余韵 | 10% |

### 4.2 每章要素

```
第X章: [章节标题]

【核心事件】
一句话概括

【情绪链条】
起始→转折→高潮→收尾

【爽点设计】
- 类型: 打脸/逆袭/反转/揭秘/情感
- 节点: 具体位置

【角色互动】
- 主角: 行动+动机
- 配角: 反应+变化
- 关系: 进展

【伏笔与钩子】
- 伏笔: 本章埋下
- 钩子: 衔接下章

【场景清单】
1. 场景A
2. 场景B
```

## 5. 情绪链条库

### 5.1 绝望剥离(虐文)

预警失灵 → 钝刀割肉 → 最后稻草 → 尸体化生存 → 消失的艺术

### 5.2 智商碾压(爽文)

猎物入笼 → 请君入瓮 → 逻辑闭环 → 公开处刑 → 视若无物

### 5.3 恐怖谷效应(悬疑)

日常裂痕 → 疯狂猜想 → 恐怖实锤 → 绝望敲门

### 5.4 扮猪吃虎(反转)

隐藏实力 → 被人挑衅 → 局部暴露 → 全面碾压

### 5.5 追妻火葬场(后悔)

轻视冷落 → 失去预警 → 疯狂追悔 → 为时已晚

## 6. 爽点设计

### 6.1 爽点类型

| 类型 | 说明 | 触发条件 |
|------|------|---------|
| 打脸爽 | 反派嘲讽→主角反击 | 反派挑衅 |
| 逆袭爽 | 低谷→崛起 | 主角受挫 |
| 反转爽 | 意料之外 | 信息差 |
| 揭秘爽 | 真相大白 | 伏笔回收 |
| 情感爽 | 情感释放 | 误会解除 |

### 6.2 爽点布局原则

- 每3-5章: 小爽点
- 每10章: 大爽点
- 高潮章节: 集中爆发
- 结尾: 终极爽点
