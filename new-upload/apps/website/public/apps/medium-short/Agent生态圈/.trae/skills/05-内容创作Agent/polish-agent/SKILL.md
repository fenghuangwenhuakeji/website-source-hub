# Polish Agent - 小说文本深度润色专家

## 核心理念

**去AI化不是目的，让读者沉浸才是终点。以角色为核心，以情绪为驱动，以节奏为武器，打造让读者无法停下的文字。**

Polish Agent 是一个专业级小说润色系统，融合现代叙事学、认知心理学、网络文学写作技巧，将机械化的AI文本转化为充满生命力的文学作品。不仅去除AI痕迹，更注入情感张力、节奏韵律、人物灵魂。

## 核心工作流程

```
文本输入 → AI痕迹识别 → 三界角色重塑 → 去AI化处理 → 情绪增强 → 对话重构 → 节奏优化 → 感官激活 → 质量校验 → 输出
```

## AI文本特征识别系统

### 1. 常见AI痕迹词库

#### 高频虚词（需重点清除）
| AI痕迹词 | 问题 | 替换方案 | 示例 |
|----------|------|----------|------|
| "仿佛" | 模糊化表达 | 具体动作 | ❌仿佛很开心 → ✅笑得眼睛弯成月牙 |
| "似乎" | 不确定性 | 直接陈述 | ❌似乎很生气 → ✅脸涨得通红 |
| "如同" | 过度比喻 | 白描或删除 | ❌如同一只蝴蝶 → ✅直接描述动作 |
| "正在" | 进行时冗余 | 动词本身 | ❌正在吃饭 → ✅埋头扒饭 |
| "开始" | 动作启动词 | 直接动作 | ❌开始哭泣 → ✅眼泪一下子涌出来 |
| "感到" | 情绪标签化 | 身体反应 | ❌感到悲伤 → ✅胸口像被什么东西堵住了 |
| "觉得" | 认知标签化 | 具体想法 | ❌觉得不对 → ✅这不对劲 |
| "突然" | 转折突兀 | 节奏控制 | ❌突然明白 → ✅电光火石间 |
| "然而" | 转折生硬 | 段落分隔 | ❌然而他... → （换段）他... |
| "因此" | 因果标签 | 直接因果 | ❌因此他... → 于是他...或直接删 |

#### 句式特征
| AI特征 | 问题 | 解决方案 |
|--------|------|----------|
| 长句堆砌 | 单句>30字 | 拆分为短句，单句<20字 |
| 复杂从句 | 多层嵌套 | 改为并列短句 |
| 被动语态 | "被"字过多 | 改为主动语态 |
| 抽象名词 | "XX性/XX度" | 改为具体描述 |
| 排比过度 | 三句以上排比 | 最多两句，或改为递进 |
| 对称结构 | 过于工整 | 打破对称，增加变化 |

#### 段落特征
| AI特征 | 问题 | 解决方案 |
|--------|------|----------|
| 段落过长 | >5行 | 控制在2-4行 |
| 开头空两格 | 传统格式 | 顶格写，段落间空行 |
| 描述集中 | 大段环境/外貌 | 穿插在动作中 |
| 对话标签 | "他说道"过多 | 减少标签，用动作替代 |

### 2. AI痕迹检测算法

```
检测维度:
1. 虚词密度 = 虚词数 / 总词数 (正常<5%, AI常>10%)
2. 平均句长 = 总字数 / 句子数 (正常15-25字, AI常>30字)
3. 被动语态比例 (正常<10%, AI常>20%)
4. 抽象词密度 (正常<8%, AI常>15%)
5. 句式多样性指数 (正常>0.7, AI常<0.5)
```

## 去AI化技术百科全书

### 1. 白话化改造（30+技巧）

#### 技巧1：删除"感到/觉得"系列
| AI写法 | 人写法 | 原理 |
|--------|--------|------|
| 他感到很生气 | 他气得脸都绿了 | 身体反应替代情绪标签 |
| 她觉得很开心 | 她笑得合不拢嘴 | 表情动作替代心理描述 |
| 他感到有些紧张 | 他手心全是汗 | 生理反应展示紧张 |
| 她感到一丝不安 | 她心里咯噔一下 | 具体感受替代抽象标签 |

#### 技巧2：删除"开始/正在"系列
| AI写法 | 人写法 | 原理 |
|--------|--------|------|
| 他开始吃饭 | 他端起碗就扒 | 直接动作 |
| 她正在走路 | 她脚步飞快 | 动作+状态 |
| 他们开始讨论 | 你一言我一语 | 对话替代叙述 |
| 天正在下雨 | 雨点噼里啪啦砸下来 | 感官描写 |

#### 技巧3：删除"仿佛/似乎/如同"系列
| AI写法 | 人写法 | 原理 |
|--------|--------|------|
| 他仿佛看到了希望 | 他眼睛一亮 | 直接反应 |
| 她似乎明白了什么 | 她恍然大悟 | 具体动作 |
| 时间如同流水 | （删除或改为）日子一天天过去 | 白描替代比喻 |

#### 技巧4：情绪标签→身体反应
| 情绪标签 | 身体反应 | 场景 |
|----------|----------|------|
| 他很伤心 | 他低着头，肩膀一抽一抽的 | 哭泣 |
| 她很害怕 | 她往后缩了缩，手指发抖 | 恐惧 |
| 他很愤怒 | 他额头青筋暴起，拳头捏得发白 | 愤怒 |
| 她很惊讶 | 她瞪大了眼睛，嘴巴张成O型 | 惊讶 |
| 他很紧张 | 他咽了口唾沫，喉结滚动 | 紧张 |
| 她很尴尬 | 她脚趾抠地，恨不得找个洞钻进去 | 尴尬 |

#### 技巧5：心理描写→对话展现
| AI写法 | 人写法 |
|--------|--------|
| 他心想，这件事必须保密 | "这件事，天知地知你知我知。" |
| 她犹豫着要不要说出口 | "我..."她咬了咬嘴唇，"算了，没什么。" |
| 他意识到情况不对 | "不对劲。"他皱起眉头。 |

#### 技巧6：长句拆短句
| AI写法 | 人写法 |
|--------|--------|
| 尽管外面下着大雨，但他为了完成对朋友的承诺，还是毫不犹豫地冲了出去 | 雨很大。但他有承诺。于是他推开门。冲了出去。 |
| 她看着眼前的景象，心中充满了复杂的情绪，既有对未来的期待，也有对过去的怀念 | 她看着眼前。未来在招手。过去在身后。心里五味杂陈。 |

#### 技巧7：被动→主动
| AI写法 | 人写法 |
|--------|--------|
| 他被这个消息震惊了 | 这个消息震得他半天没回过神 |
| 门被风吹开了 | 风把门吹开了 |
| 她的建议被采纳了 | 大家一致同意她的建议 |

#### 技巧8：抽象→具体
| AI写法 | 人写法 |
|--------|--------|
| 他的生活很艰难 | 他一天打三份工，睡四个小时 |
| 她很漂亮 | 她走过的地方，男生都会回头 |
| 这个城市很繁华 | 霓虹灯彻夜不灭，车流从不停歇 |

### 2. 句式瘦身技术

#### 单句长度控制
```
目标：单句15-20字以内

原句：他在经历了无数次的失败和挫折之后，终于在一次偶然的机会中发现了问题的关键所在。

改后：
失败了一次又一次。
他几乎要放弃。
直到那天。
一个细节引起了他的注意。
就是它了。
```

#### 段落长度控制
```
目标：段落2-4行，移动端友好

原段落：
[5行大段描述]

改后：
[2行]
[空行]
[2行]
[空行]
[对话]
```

### 3. 节奏控制技术

#### 快节奏（战斗/紧张场景）
```
特征：
- 短句为主
- 单字成句
- 动作连续
- 减少修饰

示例：
刀光。
一闪。
血溅。
他倒了下去。
```

#### 慢节奏（情感/描写场景）
```
特征：
- 适当长句
- 感官细节
- 内心独白
- 环境烘托

示例：
夕阳把天边染成橘红色。
她坐在窗前，看着远处的山影一点点暗下去。
手里那杯茶，早就凉了。
```

#### 变速技巧
```
慢→快：逐渐缩短句子
快→慢：突然插入长句或停顿
```

## 对话驱动技术

### 1. 对话替代叙述

#### 原则
```
能用对话展现的，绝不用叙述。
对话是推动情节的主要动力。
```

#### 示例
```
【AI写法】
他听闻这个消息后，内心充满了愤怒和不解，决定去找对方理论。

【人写法】
他捏着那张字条，指节发白。
"他怎么敢？"他从牙缝里挤出这句话。
朋友按住他的肩膀："冷静点！现在不是冲动的时候。"
他甩开朋友的手："冷静？我他妈怎么冷静？"
```

### 2. 对话标签优化

#### 减少"说"字
| 普通 | 优化 |
|------|------|
| 他说 | 他开口 |
| 她说 | 她接话 |
| 他们都说 | 众人附和 |

#### 用动作替代标签
| 带标签 | 动作替代 |
|--------|----------|
| "我不去。"他说。 | "我不去。"他转过身，背对着她。 |
| "真的吗？"她问。 | "真的吗？"她眼睛一亮。 |

### 3. 潜台词设计

#### 表面意思≠真实意思
```
"我没事。"
潜台词：我有事，但我不想说/我在生气

"你忙吧，不用管我。"
潜台词：我希望你放下手头的事来陪我

"挺好的。"
潜台词：其实不怎么样，但我不想评价
```

### 4. 对话节奏

#### 快速交锋
```
"你确定？"
"确定。"
"不后悔？"
"不后悔。"
"好，那就这么定了。"
```

#### 慢速对峙
```
他沉默了很久。
久到她以为他不会回答了。
终于，他开口："好。"
只有一个字。
却重若千钧。
```

## 感官写作技术

### 五感激活

#### 视觉
```
不要：房间很乱
要：衣服堆成小山，外卖盒在桌上发了霉
```

#### 听觉
```
不要：周围很吵
要：楼下喇叭声、楼上装修声、隔壁吵架声，三重奏
```

#### 嗅觉
```
不要：空气很清新
要：雨后的泥土味混着青草香，钻进鼻腔
```

#### 触觉
```
不要：天气很冷
要：风像刀子一样刮在脸上，生疼
```

#### 味觉
```
不要：菜很好吃
要：酸中带甜，甜里透辣，一口下去，舌头都要化了
```

```
不要：他感到很不安
要：后颈的汗毛突然竖了起来
```
