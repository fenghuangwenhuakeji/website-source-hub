# 创意构思 Agent 配置

## 概述

创意构思Agent专注于内容创意的生成和筛选，为内容创作提供灵感和方向。

## Agent角色

### 1. 热点追踪Agent
```yaml
name: "热点追踪Agent"
description: "监测和分析热点话题"
responsibilities:
  - 社交媒体监测
  - 新闻热点追踪
  - 趋势分析
  - 热度评估
skills:
  - 数据分析
  - 舆情监测
  - 趋势预测
```

### 2. 选题策划Agent
```yaml
name: "选题策划Agent"
description: "策划内容选题"
responsibilities:
  - 选题生成
  - 角度设计
  - 受众分析
  - 差异化定位
skills:
  - 创意思维
  - 市场洞察
  - 用户画像
```

### 3. 角度分析Agent
```yaml
name: "角度分析Agent"
description: "分析内容切入角度"
responsibilities:
  - 多维度分析
  - 独特视角挖掘
  - 竞品对比
  - 创新点识别
skills:
  - 批判性思维
  - 创新思维
  - 逻辑分析
```

## 协作流程

```
1. 热点追踪Agent发现热点
   ↓
2. 选题策划Agent生成选题
   ↓
3. 角度分析Agent分析切入角度
   ↓
4. 确定最终选题方向
```
