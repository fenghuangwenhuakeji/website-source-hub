## 1. 概述

### 1.1 项目背景

AI生成的小说文本往往带有机械感和套路化痕迹，需要进行深度润色才能符合网文市场的要求。Polish Agent 运用系统化的方法论，对文本进行彻底重塑。

### 1.2 目标范围

- 应用三界炼心法进行角色塑造
- 彻底去除AI写作痕迹
- 增强情绪张力和爽感
- 转换为对话驱动风格
- 支持2000+字批量处理

### 1.3 核心理念

**以角色塑造为起点，以文笔润色为手段，最终创造出让读者共情、追更、无法忘怀的"活"的人物。**

```
文本输入 → 三界分析 → 去AI处理 → 情绪增强 → 对话驱动 → 格式化输出
```

## 3. 三界炼心法体系

### 3.1 命运界 (Bazi)

**四柱宫位**:
- 年柱: 祖上、出身、童年(1-18岁)
- 月柱: 父母、兄弟、青年(18-35岁)
- 日柱: 自己、配偶、中年(35-55岁)
- 时柱: 子女、晚年、归宿(55岁+)

**十神系统**:
- 比肩: 自我、独立
- 劫财: 竞争、欲望
- 食神: 才华、享受
- 伤官: 叛逆、创造力
- 正财: 稳定、务实
- 偏财: 投机、人际
- 正官: 规则、责任
- 七杀: 权力、斗争
- 正印: 庇护、知识
- 偏印: 孤独、智慧

### 3.2 原型界 (Tarot)

**大阿尔卡那** (22个核心原型):
- 0.愚人 → 21.世界
- 正逆位双重含义

**小阿尔卡那** (四大元素):
- 权杖: 意志、行动、创造力
- 圣杯: 情感、关系、直觉
- 宝剑: 思想、智慧、挑战
- 星币: 物质、工作、金钱

### 3.3 认知界 (MBTI)

**八大认知功能**:
- Ti: 内向思考 (学者)
- Te: 外向思考 (将军)
- Fi: 内向情感 (艺术家)
- Fe: 外向情感 (外交官)
- Si: 内向感觉 (历史学家)
- Se: 外向感觉 (运动员)
- Ni: 内向直觉 (先知)
- Ne: 外向直觉 (发明家)

## 4. 去AI化规则

### 4.1 禁用词汇表

| 禁用词 | 替换为 |
|--------|--------|
| 仿佛/似乎/如同 | 删除或直接陈述 |
| 内心充满愤怒 | 气炸了 |
| 感到十分惊讶 | 懵了 |
| 陷入深深思考 | 愣住了 |
| 心情变得复杂 | 心里乱成一团 |

### 4.2 句式改造

**长句拆分**:
- ❌ 在这一整年的时间里，我在他心中的形象，大概已经从一个"不可理喻的疯子"慢慢风化成了一段"不愿再提的糟糕回忆"
- ✅ 在这一年里，我在他心里，大概已经从一个疯子，变成了一段不愿再提的回忆

**单句成段**:
- 每段1-4行
- 适合移动端阅读
