Place Lovart exports in this folder.

Recommended rules:

1. Keep the filenames exactly as generated from the prompt pack.
2. Preferred formats: png, webp, jpg.
3. Do not rename files after export unless necessary.
4. If multiple versions exist, keep the best one and delete the rest, or suffix with -v2, -v3.

Examples:

barb-portrait-01.png
mage-archive-01.png
startup-keyvisual-01.png
character-select-bg-01.png
settings-bg-01.png
skill-icon-whirlwind-01.png
weapon-icon-greataxe-01.png
card-art-berserk-01.png
rarity-badge-legendary-01.png

Once files are here, Codex can inspect them and wire them into the game.
