# v3.2 辅助工坊升级记录

## 改造范围
- 创意工坊
- 万能工坊
- 工具中心

## 创意工坊
- 新增「创意辅助指挥台」，显示当前工位、灵感池数量、生成模式、项目承接状态。
- 增加顶部快捷流转：批量脑洞、文本消痕、场景分镜、角色外观、平台适配、注入凤凰流。
- 灵感池加载、新增、删除、清空、存入时会同步刷新指挥台统计。
- 深度模式和批量生成数量变化后同步刷新顶部状态。

## 万能工坊
- 新增「万能工坊指挥台」，显示当前工具、子模式、链式处理状态、历史与收藏数量。
- 新增一键场景预设：章节体检、文风升级、全维拆解、素材融合、发布套装、全局联动。
- 补充小说项目常用模板：章节体检报告、章节宣发套装、小红书种草版、短视频口播版、项目全局体检、创作上下文包。
- 子模式切换后同步刷新顶部状态。

## 工具中心
- 新增「工具中心调度台」，显示当前模式、画布规模、批量处理入口、智能体入口。
- 新增顶部快捷预设：基础生成、大纲转章节、角色构建、完整流水线。
- 修正预设下拉中的乱码标签。
- 补齐原来下拉已有但未实现的工作流：循环精炼、双语翻译对比、分流融合、角色构建器。
- 加载预设、添加节点、删除节点、清空画布后同步刷新调度台统计。

## 验证
- 已对所有非 minified JS 文件执行 `node --check`，通过。
