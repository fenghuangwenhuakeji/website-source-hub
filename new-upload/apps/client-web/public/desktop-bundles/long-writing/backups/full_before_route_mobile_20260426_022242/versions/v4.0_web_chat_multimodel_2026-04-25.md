# v4.0 网页对话多模型升级

- 网页对话从单一全局 API 改为每个模型独立配置 API。
- 新增主流模型预设：OpenAI GPT、Claude、Gemini、DeepSeek、Qwen、Kimi、GLM、Doubao、Yi、Grok、Mistral、Perplexity、OpenRouter、自定义兼容接口。
- 新增模型能力标签：文本、图像输入、长上下文、推理、JSON、原生工具。
- 新增图片和文本附件输入；图像只会发给开启“图像输入”的模型。
- 新增多模型会议室：选择多个已配置模型，按轮次讨论，并可生成主持人总结。
- 配置支持系统提示词、历史窗口、温度、最大 tokens、高级请求 JSON。
- 保留旧会话数据，脚本增加版本号避免浏览器缓存旧模块。
