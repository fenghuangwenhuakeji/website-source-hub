# v3.8 移动端内容层优化

## 目标
- 保留现有移动端导航。
- 重点修复点开模块后的 PC 多栏布局挤压问题。

## 改动
- 新增 Mobile Content v2 样式层。
- 手机端 viewport 改为固定顶栏与底栏之间的独立内容区。
- 核心模块多栏布局在手机端改为单列堆叠。
- 固定宽侧栏在手机端释放为全宽顶部区域。
- 长篇执笔、凤凰创作流、世界引擎、融合拆书、拆书工作台、阅读中心、网页对话、系统设置已做移动端优先适配。
- 输入键盘弹起时自动隐藏底部导航，减少遮挡。
- 为 mobile.css、mobile_engine.js、app.js 增加版本参数，避免浏览器缓存旧移动端文件。
