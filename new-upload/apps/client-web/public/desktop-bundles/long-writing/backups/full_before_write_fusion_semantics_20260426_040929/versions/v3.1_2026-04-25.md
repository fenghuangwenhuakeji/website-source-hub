# NEXUS OS v3.1 — 创世中心 / 项目管理深度改造

> Date: 2026-04-25
> Scope: 创世中心、项目管理、入口加载稳定性

## 本次改造

### 创世中心
- 从展示型首页改为创作驾驶舱。
- 新增活跃项目指挥区：项目名、模式、阶段、进度、章节和字数一屏可见。
- 新增系统 KPI：项目数、总字数、章节数、实体数。
- 新增下一步建议：按当前项目类型推荐世界引擎、凤凰流、执笔台、拆书等入口。
- 新增系统状态：文本 API、图像 API、RAG 向量、对话会话状态。
- 新增三条创作路径卡：凤凰创作流、导入作品、拆书融合。
- 新增常用模块快速入口，减少侧边栏跳转成本。

### 项目管理
- 从项目列表升级为项目生命周期中枢。
- 新增搜索、模式筛选、状态筛选、排序、网格/列表视图切换。
- 新增活跃项目横幅，支持继续创作、保存快照、取消激活。
- 新增生命周期轨道：创建中、世界就绪、大纲完成、写作中、已完成。
- 项目卡片新增目标摘要、阶段进度、章节/字数/更新时间、激活、打开、推进阶段、重命名、删除。
- 新建项目弹窗新增创作目标字段，创建时写入项目 metadata。
- 保留原有项目快照隔离逻辑：切换项目前保存当前工作区，再加载目标项目快照。

### 稳定性修复
- 入口补加载 `ai_multimodal.js`、`theme_engine.js`、`mobile_engine.js`。
- 修正设置模块加载顺序，避免设置扩展挂不到 `Modules.settings`。
- `AI.generate` 兼容旧调用签名 `AI.generate(prompt, onChunk, config)`。
- 移动端菜单类名统一处理，兼容旧 `.open` 与新 `.mobile-open`。
