# v3.5 阅读中心 / 网页对话升级

时间：2026-04-25

## 阅读中心

- 新增顶部任务驾驶舱：导入书、继续读、问当前书、AI摘要、提取实体、存入RAG、跳转网页对话。
- 阅读中心职责收紧为“导入、阅览、提取、入库”，不再让用户在深层按钮里找功能。
- 当前书籍、藏书数、书签数、批注数集中展示，打开/关闭书籍后自动刷新状态。
- “提取实体”现在会把结果直接写入右侧阅读助手日志，并提供注入世界引擎入口。
- “存入RAG”复用已有章节识别与索引能力，把当前书拆成可检索片段。

## 网页对话

- 新增顶部任务驾驶舱：直接问、带RAG问、带项目问、续写正文、诊断文本、润色文本、设定推演、对白打磨。
- RAG和项目上下文改成明确开关，用户能直接看到当前是“纯对话”还是“带上下文对话”。
- 空会话页改成任务入口卡片，减少说明文字，直接引导用户粘贴材料拿结果。
- AI回复支持一键存入阅读中心或工作记忆，消息下方也提供单条保存入口。
- 修复流式回复更新目标不准的问题，生成中不再临时覆盖用户刚发送的消息。

## 目标

阅读中心负责看材料、整理材料；网页对话负责拿上下文交流、生成结果、回存结果。
