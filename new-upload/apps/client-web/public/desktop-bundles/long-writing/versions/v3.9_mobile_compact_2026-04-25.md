# v3.9 移动端紧凑优化

## 目标
- 在保留移动端导航的前提下，继续压缩模块内容区。
- 让手机端更像原生工作台：主内容优先，辅助操作横向滑动，减少大块面板堆叠。

## 改动
- 顶部栏、底部栏进一步变薄。
- 全局移动端按钮、卡片、间距、标题字号收紧。
- 项目管理：统计卡改为两列，生命周期改为横向滑动。
- 长篇执笔：工具按钮不再被挤成竖排，编辑区优先显示。
- 世界引擎：首屏只保留横向功能入口，隐藏低频侧栏操作。
- 拆书工作台：侧栏压缩为横向入口，主列表直接露出。
- 系统设置：横向设置入口 + 完整 quickstart 内容显示。
- 网页对话：模型选择改成横向紧凑栏，输入区减高。
- 阅读中心：书架区压缩，阅读主区优先。
- 修复一次 CSS 选择器误伤：避免把 `overflow-y-auto` 主内容区误判成宽度侧栏。
