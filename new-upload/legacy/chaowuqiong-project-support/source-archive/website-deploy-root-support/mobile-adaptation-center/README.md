# Mobile Adaptation Center

这里是 `D:\网站部署` 全部移动端适配工作的统一入口。

目标很简单：

- 不再把移动端问题零散记在聊天记录里
- 不再只改页面，不留标准
- 不再只追求“能看”，而是追求“能稳定用”

## 使用方式

1. 先看公共检查清单。
2. 再看对应项目文件。
3. 修改代码后补状态和剩余问题。
4. 构建通过后再更新报告。
