# 凤煌科技后端项目说明文档

## 项目概述

本项目是凤煌科技的后端服务系统，主要用于支持公司网站的核心功能，包括卡池系统、支付接口、用户管理和数据后台管理。项目旨在为公司提供一个稳定、安全、高效的后端服务架构，支持业务的快速发展。

## 项目目标

1. **搭建完整的后端服务架构**：基于Node.js + Express构建RESTful API服务
2. **实现核心业务功能**：卡池系统、支付接口、用户管理、订单管理
3. **提供数据后台管理**：基于Vue.js 3 + Element Plus的管理后台
4. **确保系统安全稳定**：实现身份验证、权限控制、速率限制
5. **支持平滑部署**：提供完整的部署脚本和配置方案
6. **集成第三方支付**：支付宝和微信支付SDK集成

## 功能模块

### 1. 卡池系统（Gacha System）
- **卡池管理**：创建、编辑、删除卡池
- **抽奖逻辑**：基于概率的抽奖算法
- **十连抽保证**：每十次抽奖至少获得一个高级物品
- **抽奖记录**：保存用户抽奖历史
- **物品管理**：管理卡池中物品的概率和属性

### 2. 支付系统（Payment System）
- **支付宝集成**：支持支付宝扫码支付
- **微信支付集成**：支持微信扫码支付
- **订单管理**：创建、查询、更新订单状态
- **支付回调**：处理支付平台的异步通知
- **订单验证**：确保订单的真实性和完整性

### 3. 用户系统（User System）
- **用户注册**：支持邮箱/手机号注册
- **用户登录**：基于JWT的身份验证
- **密码重置**：安全的密码重置流程
- **用户信息管理**：修改个人信息、查看历史记录
- **权限控制**：基于角色的权限管理

### 4. 管理后台（Admin Dashboard）
- **数据统计**：用户数量、订单金额、抽奖次数等数据统计
- **用户管理**：查看、编辑、禁用用户账号
- **订单管理**：查看所有订单、处理退款申请
- **卡池管理**：配置卡池参数、添加/移除物品
- **系统设置**：配置系统参数、查看系统日志

## 技术架构

### 后端技术栈
- **Node.js 18+**：运行环境
- **Express 4.18+**：Web框架
- **MySQL 8.0+**：关系型数据库
- **Redis 7.0+**：缓存系统
- **JWT**：身份验证
- **支付宝SDK**：支付宝支付集成
- **微信支付SDK**：微信支付集成

### 前端技术栈（管理后台）
- **Vue.js 3**：前端框架
- **Element Plus**：UI组件库
- **Axios**：HTTP客户端
- **Vue Router**：路由管理

### 服务器配置
- **Nginx**：反向代理、负载均衡
- **PM2**：进程管理、自动重启
- **Linux CentOS**：服务器操作系统

## 部署流程

1. **服务器准备**：
   - 安装Node.js 18+
   - 安装MySQL 8.0+
   - 安装Redis 7.0+
   - 安装Nginx
   - 安装PM2

2. **代码部署**：
   - 上传项目文件到服务器
   - 安装依赖包：`npm install`
   - 配置环境变量：`.env`文件
   - 初始化数据库：`npm run init-db`

3. **服务启动**：
   - 启动主API服务：`npm start`
   - 启动管理后台API：`npm run admin`
   - 配置PM2进程管理

4. **Nginx配置**：
   - 配置反向代理
   - 配置SSL证书（HTTPS）
   - 重启Nginx服务

## 安全考虑

1. **身份验证**：使用JWT进行身份验证，设置合理的过期时间
2. **密码安全**：使用bcryptjs对密码进行哈希处理
3. **速率限制**：实现API请求速率限制，防止暴力攻击
4. **输入验证**：使用express-validator对所有用户输入进行验证
5. **SQL注入防护**：使用参数化查询，避免SQL注入
6. **跨站请求伪造（CSRF）防护**：实现CSRF令牌验证
7. **跨域资源共享（CORS）**：合理配置CORS策略
8. **敏感信息保护**：环境变量存储敏感信息，不硬编码
