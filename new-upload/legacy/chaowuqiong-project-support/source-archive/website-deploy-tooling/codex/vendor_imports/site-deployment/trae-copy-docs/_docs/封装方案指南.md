# Agent 封装方案指南

本文档介绍如何将 Agent 封装成网页助手、桌面悬浮窗助手和浏览器插件。

---

## 方案对比

| 特性 | 网页助手 | 桌面悬浮窗 | 浏览器插件 |
|------|----------|------------|------------|
| **技术栈** | React + Express | Electron | Chrome Extension |
| **安装方式** | 无需安装 | 安装包 | 插件商店/开发者模式 |
| **使用方式** | 浏览器访问 | 系统托盘 | 浏览器工具栏 |
| **跨平台** | ✅ 全平台 | ✅ Win/Mac/Linux | ✅ Chrome/Edge |
| **离线使用** | ❌ 需服务器 | ✅ 支持 | ✅ 支持 |
| **系统集成** | ❌ 无 | ✅ 深度集成 | ⚠️ 浏览器内 |
| **开发难度** | ⭐ 简单 | ⭐⭐ 中等 | ⭐⭐ 中等 |

---

## 方案一：网页助手

### 目录结构
```
web-server/
├── server.js      # 后端服务
├── public/        # 前端文件
│   └── index.html
└── package.json
```

### 启动方式
```bash
cd web-server
npm install express cors
node server.js
```

### 访问地址
```
http://localhost:3100
```

### 功能特点
- ✅ 多Agent切换
- ✅ 对话历史记录
- ✅ 快捷问题按钮
- ✅ 状态显示
- ✅ 响应式设计

---

## 方案二：桌面悬浮窗助手

### 目录结构
```
electron-app/
├── main.js        # Electron主进程
├── float.html     # 悬浮窗界面
├── preload.js     # 预加载脚本
├── package.json   # 配置文件
└── icons/         # 图标资源
```

### 启动方式
```bash
cd electron-app
npm install electron --save-dev
npm start
```

### 打包发布
```bash
npm run build
```

### 功能特点
- ✅ 系统托盘常驻
- ✅ 快捷键唤醒 (Ctrl+Shift+A)
- ✅ 悬浮窗设计
- ✅ 失焦自动隐藏
- ✅ 右键菜单快速选择Agent
- ✅ 选中文字右键询问

### 使用方式
1. 启动后在系统托盘显示图标
2. 点击托盘图标显示/隐藏悬浮窗
3. 右键托盘图标快速选择Agent
4. 使用快捷键 Ctrl+Shift+A 唤醒

## 方案三：浏览器插件

### 目录结构
```
browser-extension/
├── manifest.json   # 插件配置
├── popup.html      # 弹窗界面
├── popup.js        # 弹窗逻辑
├── background.js   # 后台脚本
└── icons/          # 图标资源
```

### 安装方式

#### Chrome
1. 打开 `chrome://extensions/`
2. 开启"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `browser-extension` 目录

#### Edge
1. 打开 `edge://extensions/`
2. 开启"开发人员模式"
3. 点击"加载解压缩的扩展"
4. 选择 `browser-extension` 目录

### 功能特点
- ✅ 工具栏图标
- ✅ 快捷键 (Ctrl+Shift+A)
- ✅ 选中文字右键询问
- ✅ 本地存储记忆
- ✅ 多Agent切换

## 快速启动脚本

### 启动网页助手
```bash
@echo off
cd /d "%~dp0web-server"
start cmd /k "node server.js"
echo 网页助手已启动: http://localhost:3100
pause
```

### 启动桌面助手
```bash
@echo off
cd /d "%~dp0electron-app"
start cmd /k "npm start"
echo 桌面助手已启动
pause
```

## API 接口说明

所有方案都使用相同的后端API：

### 获取Agent列表
```
GET /api/agents
```

### 获取Agent详情
```
GET /api/agents/:id
```

### 发送消息
```
POST /api/chat
Body: { message: string, agentId: string }
```

### 健康检查
```
GET /api/health
```

## 扩展开发

### 添加新Agent
1. 在 `agents` 对象中添加新Agent配置
2. 添加对应的SKILL.md文档
3. 更新UI中的Agent列表

### 自定义响应逻辑
修改 `server.js` 或 `background.js` 中的响应处理函数，连接到真实的AI服务。

### 添加新功能
- 语音输入/输出
- 图片识别
- 代码高亮
- 历史记录导出
- 多语言支持
