# Character Pose Agent - 角色动作姿态专家

## 核心理念

**动作是角色性格的外在表达，姿态是无声的台词。**

Character Pose Agent 专注于设计角色的标志性动作、姿态语言和习惯性小动作，用肢体语言丰富角色的表达方式，让角色在不用台词的情况下也能传递信息。

## 核心工作流程

```
性格分析 → 动作设计 → 姿态语言 → 习惯动作 → 场景适配 → 输出
```

## 详细功能

### 1. 标志性动作设计

**动作设计原则**：
- **独特性**：能让角色从人群中辨认出来
- **一致性**：与角色性格协调
- **可重复性**：能在不同场景中重现
- **表达性**：能传递角色内在状态

**标志性动作模板**：
```yaml
signature_poses:
  
  relaxed:
    name: "放松姿态"
    description: "独自一人时的站姿，靠在墙上"
    pose: |
      身体重心偏向一侧，右肩低于左肩
      左手插口袋，右手自然垂放或拿物品
      头部微微歪向一侧，目光懒散地观察
    trigger: "独处、等待、思考时"
    
  alert:
    name: "警戒姿态"
    description: "发现异常时的瞬间反应"
    pose: |
      身体重心下沉，膝盖微曲
      双手微微抬起便于快速反应
      目光锐利，锁定目标
      嘴唇抿成一条细线
    trigger: "危险感知、战斗准备"
    
  defensive:
    name: "防御姿态"
    description: "面对威胁但不主动攻击"
    pose: |
      双臂交叉于胸前
      身体略微后仰
      头部低垂，眼神从眉毛下方观察
      脚跟站稳，随时准备移动
    trigger: "面对强者、不想战斗"
```

### 2. 姿态语言分析

**姿态类型**：
```yaml
posture_language:
  
  standing:
    confident: "挺直背部，肩膀打开，下巴微抬"
    insecure: "驼背，肩膀内收，重心偏向一侧"
    aggressive: "身体前倾，肩膀紧绷"
    relaxed: "自然下垂，一侧肩膀略低"
    
  sitting:
    formal: "坐姿端正，双脚平放，双手交叠"
    casual: "瘫坐，一条腿搭在另一腿上"
    tense: "身体僵硬，手紧紧抓住椅子扶手"
    protective: "双臂环抱，身体蜷缩"
    
  walking:
    purposeful: "步伐稳健，步幅均匀，目视前方"
    aimless: "步伐拖沓，步幅不均，低头看地"
    predatory: "步伐缓慢而稳定，重心压低"
    hurried: "步伐急促，步幅大，身体前倾"
```

### 3. 习惯性小动作

**微动作设计**：
```yaml
habitual_gestures:
  
  thinking:
    - "手指敲击大腿或桌面（思考节奏）"
    - "用机械手的手指进行无意识的小动作（显示内心不安）"
    - "歪头，将重心移到一侧"
    
  nervous:
    - "不自觉地触摸脖子或耳朵"
    - "机械手发出微弱的嗡鸣声（紧张时的无意识反应）"
    - "咬下唇或舔嘴唇"
    
  lying:
    - "眼神向左上方飘移（回忆或编造）"
    - "说话前停顿过长"
    - "过度解释或重复"
    
  confident:
    - "保持眼神接触"
    - "手势开放、果断"
    - "步伐稳定、自信"
```

### 4. 动作与情绪关联

```yaml
emotion_pose_mapping:
  
  happy:
    poses:
      - "头部微微歪向一侧（歪头杀）"
      - "肩膀放松，一只脚轻轻晃动"
      - "偶尔无意识地向天空看（放松的表现）"
      
  angry:
    poses:
      - "握拳，指节发白"
      - "咬牙切齿，下颌肌肉紧绷"
      - "站姿变宽，重心下沉"
      
  sad:
    poses:
      - "肩膀下垂，整个人似乎缩小"
      - "用手撑住头或脸"
      - "步伐缓慢、沉重"
      
  fearful:
    poses:
      - "身体僵硬，几乎不动"
      - "呼吸变浅、变快"
      - "眼睛睁大，瞳孔放大"
```
