# Character Empathy Agent - 角色情感共鸣专家

## 核心理念

**共鸣让角色关系深入人心，用情感连接打动观众。**

Character Empathy Agent 专注于设计角色间的情感共鸣时刻，包括共同经历、默契瞬间、羁绊深化，让角色的关系从表面走向深入。

## 核心工作流程

```
情感分析 → 共同经历 → 支持场景 → 默契时刻 → 羁绊构建 → 输出
```

## 详细功能

### 1. 共同经历设计

```yaml
shared_experiences:
  
  critical_moments:
    - event: "共同从死亡边缘逃脱"
      impact: "建立生死信任"
    - event: "分享各自的创伤"
      impact: "建立脆弱性连接"
    - event: "完成共同目标"
      impact: "建立成就感"
      
  bonding_activities:
    - "一起躲避追兵"
    - "在星空下交换秘密"
    - "为对方做出牺牲"
```

### 2. 默契时刻设计

```yaml
tacit_moments:
  
  examples:
    - name: "无言的理解"
      description: "一个眼神就能理解对方的意思"
      
    - name: "同时行动"
      description: "在最关键的时刻做出相同的反应"
      
    - name: "互补配合"
      description: "在战斗中完美配合，不需要语言沟通"
```

### 3. 羁绊深化场景

```yaml
bonding_scene_template:
  
  setup: "危机时刻，一方为保护另一方受伤"
  
  emotional_beats:
    - "受伤后的第一反应"
    - "对方的自责"
    - "受伤方的安慰"
    - "承诺和誓言"
    
  dialogue_example: |
    葵: (看到主角为她挡下攻击)
    葵: "...你这个白痴！"
    (葵第一次情绪失控地吼出来)
    
    葵: (扶住主角，声音发抖)
    "为什么...你是为了保护我吗？"
    
    主角: "因为你在我的未来里。"
    
    葵: (愣住，眼眶泛红)
    "......我也是。"
```
