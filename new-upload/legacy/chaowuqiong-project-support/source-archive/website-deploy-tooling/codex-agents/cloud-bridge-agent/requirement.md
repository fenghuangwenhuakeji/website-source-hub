# Requirements

## Primary Outcome

Provide a reusable local operating package for:

- SSH bridge connection
- cloud deployment
- post-deploy verification
- later MySQL separation

## Functional Requirements

- keep a documented known-good SSH path
- keep the reusable bridge scripts in one folder
- preserve root-level compatibility entry points
- keep deployment history with absolute dates
- keep MySQL separation guidance next to deployment tooling

## Non-Functional Requirements

- minimal operator memory load
- safe rollback awareness
- easy handoff to future cloud and database work
