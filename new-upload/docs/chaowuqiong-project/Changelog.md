# 超无穹AI 变更日志 (Changelog)

## [1.0.0] - 2024-03-27

### 新增功能

#### 用户系统
- 用户注册/登录功能
- JWT Token认证
- 用户信息管理
- 密码修改功能

#### 会员系统
- 8种会员套餐支持
- 会员时长管理
- 会员权益系统
- 积分赠送机制

#### 支付系统
- 支付宝扫码支付集成
- 订单管理系统
- 支付状态回调
- 模拟支付模式

#### 积分系统
- 积分余额管理
- 积分消费记录
- 积分获取记录
- 积分统计展示

#### 邀请系统
- 邀请码生成
- 邀请关系绑定
- 邀请奖励发放
- 邀请里程碑奖励
  - 邀请3人送周卡
  - 邀请10人送月卡

#### AI功能
- 多模型AI对话
- API池管理
- 自定义API配置
- 模型切换功能

#### 桌面客户端
- Electron桌面应用
- Windows/macOS/Linux支持
- 自动更新功能
- 系统托盘集成

### 优化改进

#### 移动端适配
- 全局响应式样式系统
- ChatPanel移动端优化
- RechargeCenter移动端优化
- LoginGate移动端优化
- 安全区域适配

#### 性能优化
- 前端代码分割
- 图片懒加载
- API响应缓存
- 数据库查询优化

### 技术架构

#### 前端
- React 18 + TypeScript
- Vite构建工具
- SCSS Modules样式
- Context状态管理

#### 后端
- Node.js + Express
- TypeScript
- MySQL 8.0
- Redis缓存

#### 部署
- Nginx反向代理
- PM2进程管理
- SSL证书配置
- 自动化部署脚本
