## 🎮 如何运行

### 方法1：直接打开
1. 双击 `index.html` 文件
2. 在浏览器中直接运行

### 方法2：本地服务器（推荐）
```bash
# 使用Python
python -m http.server 8000

# 使用Node.js
npx http-server

# 使用PHP
php -S localhost:8000
```
然后访问 `http://localhost:8000/index.html`

## 🎯 游戏流程

1. **创建角色**
   - 输入角色名称
   - 选择职业
   - 开始游戏

2. **序章剧情**
   - 了解游戏背景
   - 学习基本操作
   - 接取初始任务

3. **探索世界**
   - 在地图上移动
   - 探索不同地点
   - 遭遇怪物战斗

4. **角色成长**
   - 战斗获得经验
   - 升级学习技能
   - 收集卡牌装备

5. **完成任务**
   - 接取任务
   - 完成目标
   - 获得奖励

6. **转职进阶**
   - 转换职业
   - 学习新技能
   - 挑战更强大的敌人
