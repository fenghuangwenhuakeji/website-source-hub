## 💡 技术亮点

1. **模块化架构** - 清晰的模块划分，便于维护和扩展
2. **事件驱动** - 使用事件系统实现松耦合的系统间通信
3. **状态管理** - 统一的状态管理，支持订阅和快照
4. **响应式设计** - 支持多种屏幕尺寸
5. **数据驱动** - 游戏内容与逻辑分离，便于修改
6. **ES6模块** - 使用现代JavaScript特性

- **前端**: HTML5, CSS3, JavaScript ES6+
- **模块化**: ES6 Modules
- **渲染**: Canvas API
- **存储**: LocalStorage
- **构建**: 原生（无需构建工具）
