# 统一酒馆游戏 - 快速启动指南

## 🚀 5分钟快速开始

### 第1步：打开游戏（1分钟）
直接用浏览器打开 `index.html` 文件即可开始游戏。

### 第2步：创建角色（2分钟）
1. 点击游戏大厅的 "新游戏" 按钮
2. 输入你的角色名称（例如：勇者小明）
3. 选择种族：
   - **人类**：平衡发展，适合新手
   - **精灵**：高智力，适合法师
   - **矮人**：高体质，适合战士
4. 选择职业：
   - **战士** ⚔️ - 高生命高防御，近战专家
   - **法师** 🔮 - 高魔法值，远程魔法攻击
   - **盗贼** 🗡️ - 高速度高暴击，快速攻击
   - **牧师** ✨ - 治疗支援，神圣力量
   - **弓箭手** 🏹 - 远程精准，陷阱技能
5. 点击确认，开始冒险！

### 第3步：探索酒馆（1分钟）
1. 点击左侧导航的 "酒馆" 按钮
2. 与NPC对话，了解游戏世界
3. 查看任务板，接取第一个任务
4. 在商店购买初始装备

### 第4步：开始战斗（1分钟）
1. 点击 "地图探索" 按钮
2. 选择一个地点（推荐：森林）
3. 点击移动，可能遭遇怪物
4. 进入战斗，使用卡牌和技能击败敌人

## 🎮 基础操作

### 鼠标操作
- **左侧导航**: 点击切换不同视图
- **角色面板**: 查看角色属性和状态
- **地图系统**: 点击地点图标移动
- **战斗界面**: 点击卡牌或技能按钮行动
- **任务列表**: 点击查看任务详情

### 键盘快捷键
```
1 - 地图视图
2 - 酒馆视图
3 - 角色面板
4 - 卡牌收藏
5 - 职业系统
ESC - 打开菜单
S - 保存游戏
L - 读取游戏
P - 暂停/继续
```

## 💡 新手建议

### 开局建议
1. **选择战士或法师**：这两个职业对新手最友好
2. **先完成新手任务**：快速获得经验和装备
3. **不要远离村庄**：初期在村庄附近探索
4. **及时保存游戏**：避免意外丢失进度

### 战斗技巧
1. **合理使用卡牌**：根据情况选择攻击或防御
2. **管理好MP**：不要一次性用完所有MP
3. **注意敌人状态**：眩晕、中毒等状态可以帮你战斗
4. **及时恢复**：战斗前确保生命值充足

### 升级策略
1. **优先升级主属性**：战士加力量/体质，法师加智力
2. **学习核心技能**：优先学习职业招牌技能
3. **收集强力卡牌**：稀有和史诗卡牌更强
4. **升级装备**：定期更新武器和护甲

## 🛠️ 常见问题

### Q1: 游戏打不开？
**A**: 确保用现代浏览器（Chrome、Firefox、Edge）打开index.html文件。

### Q2: 存档丢失了？
**A**: 游戏使用LocalStorage存储存档，清理浏览器缓存会清除存档。建议定期导出存档。

### Q3: 战斗太困难？
**A**:
- 完成更多任务获得奖励
- 升级角色和装备
- 收集更好的卡牌
- 使用调试命令：`Game.addExp(1000)` 快速升级

### Q4: AI功能不工作？
**A**: AI功能需要配置API密钥。默认使用模拟数据，实际使用时需要：
1. 在控制台输入：`localStorage.setItem('ai_api_key', 'your-api-key')`
2. 刷新页面
3. AI功能将使用真实API

### Q5: 如何恢复默认设置？
**A**: 在控制台输入：
```javascript
localStorage.clear();
location.reload();
```

## 📚 游戏指南

### 职业详解

#### 战士 ⚔️
- **特点**: 高生命、高防御、近战
- **适合**: 喜欢正面战斗的玩家
- **核心技能**: 斩击、盾击、战吼
- **推荐加点**: 力量 > 体质 > 敏捷

#### 法师 🔮
- **特点**: 高魔法、远程攻击、爆发力强
- **适合**: 喜欢策略和控场的玩家
- **核心技能**: 火球术、冰刺、闪电链
- **推荐加点**: 智力 > 体质 > 敏捷

#### 盗贼 🗡️
- **特点**: 高速度、高暴击、潜行
- **适合**: 喜欢快速和暴击的玩家
- **核心技能**: 背刺、涂毒、潜行
- **推荐加点**: 敏捷 > 力量 > 幸运

#### 牧师 ✨
- **特点**: 治疗、支援、神圣力量
- **适合**: 喜欢辅助和团队的玩家
- **核心技能**: 治疗、祝福、神圣打击
- **推荐加点**: 智力 > 体质 > 幸运

#### 弓箭手 🏹
- **特点**: 远程、精准、陷阱
- **适合**: 喜欢风筝和控制的玩家
- **核心技能**: 瞄准射击、箭雨、陷阱
- **推荐加点**: 敏捷 > 力量 > 体质

### 卡牌攻略

#### 卡牌稀有度
- **普通** (灰色): 基础卡牌，容易获得
- **稀有** (蓝色): 较强效果，需要一定运气
- **史诗** (紫色): 强大效果，稀有掉落
- **传说** (金色): 极强效果，BOSS掉落

#### 卡组构建
1. **攻击卡** (40-50%): 造成伤害
2. **防御卡** (20-30%): 保护自己
3. **功能卡** (20-30%): 特殊效果

#### 推荐卡组
**战士卡组**: 斩击卡、重击、双重打击、防御、钢铁意志
**法师卡组**: 火球、冰爆、魔法卡牌、防御、治疗
**盗贼卡组**: 斩击卡、双重打击、防御、涂毒、潜行

### 地图攻略

#### 新手村 🏘️
- **等级要求**: 1-3级
- **怪物**: 无
- **功能**: 休息、交易、接任务
- **提示**: 初期安全区域，可以随时回来

#### 传奇酒馆 🏰
- **等级要求**: 1级+
- **怪物**: 无
- **功能**: NPC对话、商店、任务板
- **提示**: 主要任务接取点

#### 迷雾森林 🌲
- **等级要求**: 1-5级
- **怪物**: 史莱姆、哥布林、狼
- **功能**: 探索、战斗
- **提示**: 新手练级的好地方

#### 黑暗洞穴 🕳️
- **等级要求**: 3-8级
- **怪物**: 骷髅、哥布林
- **功能**: 探索、挖掘
- **提示**: 小心骷髅的召唤技能

#### 龙之山脉 ⛰️
- **等级要求**: 8-15级
- **怪物**: 狼、兽人、幼龙
- **功能**: 攀登、狩猎
- **提示**: 幼龙是BOSS，做好充分准备

#### 古代遗迹 🏛️
- **等级要求**: 5-12级
- **怪物**: 骷髅、兽人
- **功能**: 探索、挖掘
- **提示**: 可能发现稀有装备

## 🎓 进阶技巧

2. **效率路线**: 规划最优探索路线
3. **奖励最大化**: 优先完成奖励高的任务
