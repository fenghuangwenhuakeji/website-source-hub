# 🎮 传奇酒馆 - 快速启动指南

## 🚀 30秒开始游戏

### 步骤1：打开游戏
```
双击 index.html 文件
```

### 步骤2：创建角色
```
1. 输入角色名称（例如：亚瑟）
2. 选择职业（1-战士，2-法师，3-盗贼，4-牧师，5-弓箭手）
3. 点击确定
```

### 步骤3：开始冒险
```
观看序章剧情 → 接取任务 → 探索地图 → 战斗升级
```

## 🎯 新手指南

### 第一次战斗
1. 点击地图上的森林或洞穴
2. 遭遇怪物后进入战斗
3. 选择「攻击」进行普通攻击
4. 消耗MP使用「技能」
5. 消耗能量打出「卡牌」
6. 击败怪物获得经验和金币

### 升级角色
1. 积累足够经验值会自动升级
2. 进入「角色面板」查看属性
3. 在「升级系统」分配技能点
4. 学习新技能提升战斗力

### 收集卡牌
1. 战斗胜利有机会获得卡牌
2. 在「卡牌收藏」查看所有卡牌
3. 不同类型卡牌有不同效果
4. 构建强力卡组

### 完成任务
1. 在酒馆查看「任务板」
2. 接取任务后查看目标
3. 完成目标获得奖励
4. 任务奖励包括经验、金币、道具

## ⚔️ 职业选择

### 战士 ⚔️
- **特点**：高生命、高防御
- **适合**：近战冲锋、坦克角色
- **技能**：斩击、盾击、战吼

### 法师 🔮
- **特点**：高魔法、远程攻击
- **适合**：魔法输出、群体伤害
- **技能**：火球术、冰刺、闪电链

### 盗贼 🗡️
- **特点**：高速度、高暴击
- **适合**：快速击杀、爆发输出
- **技能**：背刺、涂毒、潜行

### 牧师 ✨
- **特点**：治疗、支援
- **适合**：团队支援、持续战斗
- **技能**：治疗、祝福、神圣打击

### 弓箭手 🏹
- **特点**：远程、精准
- **适合**：安全输出、风筝战术
- **技能**：瞄准射击、箭雨、陷阱

## 🔧 常见问题

### Q: 如何保存游戏？
A: 按 ESC 打开菜单，选择「保存游戏」。游戏也会每5分钟自动保存。

### Q: 如何恢复生命和魔法？
A: 在酒馆选择「住宿」休息，或者使用生命药水和魔法药水。

### Q: 战斗失败会怎样？
A: 不会丢失进度，但生命值会减半，金币会减少10%。

### Q: 如何转职？
A: 在「职业系统」中选择其他职业，转职会改变技能和属性加成。

### Q: 卡牌如何获得？
A: 战斗胜利、完成任务、商店购买都有机会获得卡牌。

### Q: 技能点不够怎么办？
A: 升级会获得技能点，每级2点。学习新技能需要3点。

### Q: 怎么变强？
A: 
1. 升级提升基础属性
2. 学习强力技能
3. 收集优质装备
4. 构建强力卡组
5. 完成任务获得奖励
