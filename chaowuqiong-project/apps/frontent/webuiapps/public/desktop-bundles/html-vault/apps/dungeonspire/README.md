# DungeonSpire

A massive, data-driven Roguelike deckbuilder engine.

## Features
- 500+ generated data files
- 6 Playable Classes (Warrior, Mage, Rogue, Warlock, Paladin, Druid)
- Deep Item & Crafting System
- Modding Support
- Localization (EN, ZH, JP, FR, DE)

## Setup
1. Clone repository
2. Run `npm install`
3. Run `npm start`
