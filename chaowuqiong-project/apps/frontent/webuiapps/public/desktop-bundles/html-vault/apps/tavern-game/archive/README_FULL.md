# AI酒馆 RPG - 完整版

## 🎮 游戏简介

这是一个融合了人工智能生成技术的完整角色扮演游戏（RPG）系统，基于Web技术构建，包含完整的八大核心系统。

## ✨ 核心特性

### 1. 🎨 AI生成内容系统
- **文本生成**：使用大语言模型生成剧情、对话、描述
- **图像生成**：生成角色头像、场景图像、物品图标
- **语音合成**：文本转语音，为角色配音
- **视频生成**：生成场景视频片段
- **智能缓存**：自动缓存生成内容，提高性能

### 2. 🗺️ 地图系统
- **交互式世界地图**：2000x1500像素的广阔世界
- **多地图支持**：世界地图、室内地图、地下城地图
- **迷雾系统**：战争迷雾，探索未知区域
- **路径导航**：自动计算最优路线
- **地点发现**：探索过程中自动发现新地点
- **自定义地图**：支持创建和导入自定义地图

### 3. 💬 交互系统
- **深度NPC对话**：基于性格和好感度的智能对话
- **多话题系统**：每个NPC有多个可探索的话题
- **好感度系统**：影响对话内容和交易价格
- **服务系统**：NPC提供各种服务（交易、修理、休息等）
- **历史记录**：完整的交互历史追踪

### 4. ⚔️ 战斗系统
- **回合制战斗**：经典的回合制RPG战斗
- **多敌人战斗**：同时对抗多个敌人
- **技能系统**：丰富的技能库和技能组合
- **状态效果**：增益、减益、持续伤害等
- **BOSS战**：多阶段BOSS战斗
- **战斗AI**：不同敌人有不同的AI策略

### 5. 🃏 卡牌系统
- **完整卡库**：50+张不同类型的卡牌
- **卡组构建**：自由组合卡组
- **卡牌升级**：卡牌可以升级强化
- **能量系统**：每回合有限能量，策略出牌
- **稀有度系统**：普通、罕见、稀有、史诗、传说
- **职业卡牌**：每个职业有专属卡牌

### 6. 📖 剧情系统
- **分支剧情**：玩家选择影响故事走向
- **多结局**：多个可能的结局
- **条件触发**：基于玩家属性和行为的剧情分支
- **章节系统**：分章节的主线剧情
- **选择历史**：记录所有关键决策

### 7. 📈 角色升级系统
- **等级系统**：经验值升级，属性成长
- **属性系统**：六大属性（力量、敏捷、智力、体质、幸运、魅力）
- **衍生属性**：HP、MP、攻击力、防御力等
- **技能点分配**：自由分配技能点
- **种族加成**：不同种族有不同的属性加成

### 8. ⚔️ 职业系统
- **基础职业**：战士、法师、盗贼
- **转职系统**：20级可转职为子职业
- **技能树**：每个职业有独特的技能树
- **被动技能**：永久提升属性或能力
- **职业特性**：每个职业有独特的战斗风格

## 🚀 快速开始

### 环境要求
- 现代浏览器（Chrome、Firefox、Edge、Safari）
- 支持ES6模块
- 本地服务器（推荐）或直接打开HTML文件

### 安装步骤

1. **克隆或下载项目**
```bash
git clone [repository-url]
cd AI_Tavern_Refactored
```

2. **使用本地服务器启动**

方法1：使用Python
```bash
python -m http.server 8000
```

方法2：使用Node.js
```bash
npx http-server -p 8000
```

方法3：使用VS Code Live Server扩展
- 安装Live Server扩展
- 右键点击index_full.html
- 选择"Open with Live Server"

3. **打开游戏**
在浏览器中访问 `http://localhost:8000/index_full.html`

### 直接打开（无服务器）
也可以直接在浏览器中打开 `index_full.html` 文件，但某些功能可能受限。

## 🎯 游戏操作

### 基础操作
- **移动**：在地图上点击移动
- **交互**：点击NPC或物体进行交互
- **对话**：与NPC对话，选择话题
- **战斗**：使用卡牌或技能进行战斗

### 快捷键
- `Enter` - 发送输入
- `Escape` - 关闭弹窗
- `M` - 打开地图
- `I` - 打开背包
- `C` - 打开角色面板
- `S` - 保存游戏
- `L` - 读取游戏

## 🔧 系统配置

### API配置
游戏使用AI生成内容，需要配置API密钥：

1. 打开游戏
2. 进入"API设置"
3. 添加你的API配置
4. 选择服务商（Gemini、OpenAI、Claude等）
5. 输入API Key
6. 保存配置

支持的API服务商：
- Google Gemini
- OpenAI (GPT-4)
- Anthropic Claude
- DeepSeek
- 通义千问
- 自定义/OpenAI兼容API

### 游戏设置
- **音效音量**：调节音效音量
- **音乐音量**：调节背景音乐音量
- **自动保存**：启用/禁用自动保存
- **自动保存间隔**：设置自动保存间隔
- **游戏速度**：调节游戏速度

## 🎮 游戏系统详解

### 角色创建
1. 点击"新游戏"
2. 输入角色名称
3. 选择职业（战士、法师、盗贼）
4. 选择种族（人类、精灵、矮人、兽人、龙人、地精）
5. 选择性别
6. 确认创建

### 探索系统
- 点击地图上的位置进行移动
- 探索时会自动发现附近的地点
- 随机遭遇敌人和事件
- 与NPC交互获取信息和任务

### 战斗系统
- 进入战斗后，抽取手牌
- 使用卡牌攻击敌人或防御
- 消耗能量打出卡牌
- 每回合结束后抽新牌
- 击败敌人获得经验值和奖励

### 升级系统
- 击败敌人获得经验值
- 经验值满后升级
- 升级获得属性点和技能点
- 分配点数强化角色

### 职业系统
- 20级可转职
- 每个职业有多个子职业可选
- 子职业继承并扩展主职业技能
- 转职后获得新的技能树

## 🎨 自定义内容

### 创建自定义地图
```javascript
// 在游戏控制台中或通过API
const customMap = {
    id: 'my_map',
    name: '自定义地图',
    type: 'custom',
    size: { width: 1000, height: 800 },
    locations: [
        // 添加地点
    ]
};

gameEngine.mapSystem.createMap(customMap);
```

### 创建自定义卡牌
```javascript
const customCard = {
    id: 'my_card',
    name: '我的卡牌',
    type: 'attack',
    cost: 2,
    rarity: 'rare',
    description: '造成15点伤害',
    effect: 'damage',
    value: 15,
    icon: '⭐'
};

gameEngine.cardSystem.addCard(customCard);
```

### 创建自定义NPC
```javascript
const customNPC = {
    id: 'my_npc',
    name: '新NPC',
    title: '神秘旅人',
    icon: '🎭',
    personality: 'friendly',
    dialogue: {
        greeting: ['你好！'],
        topics: []
    }
};

gameEngine.interactionSystem.addNPC(customNPC);
```

## 💾 存档系统

### 自动保存
- 默认每5分钟自动保存
- 可在设置中调整间隔
- 保存位置：localStorage

### 手动保存
- 点击顶部"保存"按钮
- 或按快捷键S
- 可选择存档槽位

### 存档内容
- 角色数据
- 游戏进度
- 所有系统状态
- 地图探索进度
- 剧情决策历史

## 🐛 故障排除

### 常见问题

**Q: 游戏无法加载**
A: 确保使用现代浏览器，并启用了JavaScript和LocalStorage

**Q: AI生成功能不工作**
A: 检查API配置是否正确，网络连接是否正常

**Q: 存档丢失**
A: 清除浏览器缓存会删除存档，建议定期导出存档

**Q: 游戏运行卡顿**
A: 关闭其他标签页，降低游戏画面质量

### 调试模式
在控制台中输入：
```javascript
gameEngine.getGameState()    // 查看游戏状态
gameEngine.getSystemStates() // 查看系统状态
```
