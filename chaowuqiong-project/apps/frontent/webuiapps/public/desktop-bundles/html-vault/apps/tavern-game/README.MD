# 统一酒馆游戏 - UNIFIED TAVERN GAME

> 🏰 融合AI生成技术的完整角色扮演游戏系统

## 项目简介

统一酒馆游戏是将两个优秀的酒馆RPG项目（AI_Tavern_Refactored 和 RPG_TAVERN_GAME）合并而成的一个完整、强大、可扩展的游戏系统。

### 项目背景

- **RPG_TAVERN_GAME** - 提供完整的RPG核心系统（角色、战斗、卡牌、剧情、任务）
- **AI_Tavern_Refactored** - 提供AI内容生成、模块化架构、高级UI交互
- **统一整合** - 取两者之长，打造完整的Web RPG游戏系统

### 核心特色

- ✨ **完整角色系统** - 创建、升级、转职、技能树，5种职业+10+转职
- ⚔️ **深度战斗系统** - 回合制战斗 + 卡牌系统 + 技能组合
- 🗺️ **广阔地图探索** - 2000x1500像素世界、多地点、随机事件、迷雾系统
- 🏰 **丰富酒馆互动** - NPC对话、商店、任务、住宿、好感度系统
- 🎨 **AI内容生成** - 动态对话、场景、角色、物品图像生成
- 📖 **分支剧情系统** - 多结局、选择影响故事走向
- 🃏 **卡牌收集** - 50+张卡牌，卡组构建，稀有度系统
- 📜 **任务系统** - 主线、支线、日常任务，任务追踪
- 💾 **完整存档** - 多存档位、自动保存、导入导出

## 快速开始

### 环境要求

- 现代浏览器（Chrome、Firefox、Edge、Safari）
- 支持ES6模块
- 本地服务器（推荐）或直接打开HTML文件

### 安装和运行

#### 方法1：直接打开（最简单）

1. 下载或克隆项目
```bash
git clone [repository-url]
cd TAVERN_GAME
```

2. 直接在浏览器中打开 `index.html`

#### 方法2：使用本地服务器（推荐）

**使用Python：**
```bash
python -m http.server 8000
```

**使用Node.js：**
```bash
npx http-server -p 8000
```

**使用项目提供的脚本：**

Windows:
```bash
启动服务器.bat
```

Linux/Mac:
```bash
./start-server.sh
```

或使用Python:
```bash
python start.py
```

3. 在浏览器中访问 `http://localhost:8000`

## 游戏操作

### 基础操作
- **移动**：在地图上点击位置移动
- **交互**：点击NPC或物体进行交互
- **对话**：与NPC对话，选择话题
- **战斗**：使用卡牌或技能进行战斗

### 快捷键

| 键位 | 功能 |
|------|------|
| `1` / `M` | 地图视图 |
| `2` | 酒馆视图 |
| `3` / `C` | 角色面板 |
| `4` | 卡牌收藏 |
| `5` | 职业系统 |
| `S` | 保存游戏 |
| `L` | 读取游戏 |
| `ESC` | 打开菜单/关闭弹窗 |
| `Enter` | 确认/发送 |

## 核心系统详解

### 1. 角色系统
- **角色创建**：姓名、种族、职业、初始属性
- **属性系统**：
  - 六大基础属性：力量(STR)、敏捷(AGI)、智力(INT)、体质(CON)、幸运(LUCK)、魅力(CHA)
  - 衍生属性：HP、MP、攻击力、防御力、暴击率等
- **职业系统**：5种基础职业（战士/法师/盗贼/牧师/弓箭手）
- **转职系统**：Lv.20可转职为2-3个方向
- **技能树**：每个职业有独特的技能树

### 2. 战斗系统
- **回合制战斗**：基于速度的行动顺序
- **卡牌系统**：战斗中使用卡牌发动技能和效果
- **技能系统**：丰富的技能库，技能组合连招
- **状态效果**：增益、减益、持续伤害（眩晕、中毒、燃烧等）
- **BOSS战**：多阶段战斗，特殊机制
- **战斗AI**：不同敌人有不同的AI策略

### 3. 地图系统
- **广阔世界**：2000x1500像素的Canvas渲染地图
- **多地图支持**：世界地图、城市地图、室内地图、地下城
- **迷雾系统**：未探索区域不可见
- **路径导航**：点击自动移动到目标位置
- **随机遭遇**：探索时可能触发战斗或事件
- **地点系统**：多个可探索地点（村庄、酒馆、森林、洞穴、山脉、遗迹）

### 4. 酒馆系统
- **NPC交互**：与酒馆老板、商人、旅行者等对话
- **好感度系统**：影响对话内容和交易价格
- **商店功能**：购买装备、物品、出售多余物品
- **休息系统**：恢复HP和MP
- **任务板**：接取新的冒险任务
- **消息服务**：获取游戏情报和线索

### 5. 剧情系统
- **分章节**：主线剧情分为多个章节
- **分支选择**：玩家选择影响剧情走向
- **多结局**：多个可能的结局
- **角色发展**：剧情中角色的成长和变化
- **AI生成**：动态生成对话和场景描述（可选）

### 6. 卡牌系统
- **卡牌类型**：攻击、防御、魔法、辅助
- **稀有度**：普通、罕见、稀有、史诗、传说
- **卡组构建**：最多20张卡牌的自由组合
- **卡牌升级**：提升卡牌效果
- **能量系统**：每回合有限能量，策略出牌
- **职业卡牌**：每个职业有专属卡牌

### 7. 任务系统
- **任务类型**：主线、支线、日常、成就
- **任务追踪**：实时显示任务进度
- **奖励系统**：金币、经验、物品、装备
- **任务板**：在酒馆接取新任务
- **自动导航**：提示任务目标位置

### 8. AI内容生成（可选功能）
- **文本生成**：生成NPC对话、场景描述、物品描述
- **图像生成**：生成角色头像、场景图片、物品图标
- **语音合成**：文本转语音，为角色配音
- **智能缓存**：自动缓存生成内容，提高性能

### 9. 存档系统
- **多存档位**：5个存档位
- **自动保存**：关键点自动保存
- **手动保存**：玩家主动保存
- **导入/导出**：跨设备存档共享
- **存档管理**：查看存档详情、删除存档

## 常见问题

### Q: 游戏无法保存？
A: 请确保浏览器允许使用LocalStorage。如果使用无痕模式，可能无法保存。

### Q: 地图显示不正常？
A: 推荐使用本地服务器运行，直接打开HTML文件可能会有跨域问题。

### Q: AI功能如何启用？
A: AI功能需要配置API密钥。默认情况下游戏会使用预生成的内容，无需API密钥即可游玩。

### Q: 如何重置游戏？
A: 在浏览器控制台运行：
```javascript
localStorage.clear();
location.reload();
```
