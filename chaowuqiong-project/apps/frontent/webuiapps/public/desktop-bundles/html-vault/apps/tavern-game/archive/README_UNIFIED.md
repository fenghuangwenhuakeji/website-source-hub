# 统一酒馆游戏 (UNIFIED TAVERN GAME)

## 🎮 项目简介

这是一个融合了 **RPG_TAVERN_GAME** 和 **AI_Tavern_Refactored** 的统一酒馆游戏项目，集成了两个项目的所有优势功能：

### RPG_TAVERN_GAME 的特色
- ✅ 完整的RPG角色系统（5种职业）
- ✅ 深度战斗系统（回合制战斗）
- ✅ 卡牌收集和构建系统
- ✅ 丰富的游戏数据（职业、技能、卡牌、怪物、物品、地图、NPC、任务）
- ✅ 剧情系统（分支剧情、多结局）
- ✅ 任务系统（主线、支线、每日任务）
- ✅ 存档系统（多存档位、自动保存、导入导出）

### AI_Tavern_Refactored 的特色
- ✅ AI内容生成系统（文本、图像、语音）
- ✅ 模块化架构设计
- ✅ 高级UI组件和动画
- ✅ 交互系统（NPC对话）
- ✅ 工作流管理
- ✅ RAG知识库集成

## 🚀 快速开始

### 1. 启动游戏

直接用浏览器打开 `index.html` 文件即可开始游戏。

### 2. 创建角色

1. 点击 "新游戏" 按钮
2. 输入角色名称
3. 选择种族（人类/精灵/矮人）
4. 选择职业（战士/法师/盗贼/牧师/弓箭手）
5. 开始冒险！

### 3. 游戏操作

#### 键盘快捷键
- `1` - 地图视图
- `2` - 酒馆视图
- `3` - 角色面板
- `4` - 卡牌收藏
- `5` - 职业系统
- `ESC` - 打开菜单

#### 鼠标操作
- 点击左侧导航切换视图
- 点击地图图标进行移动
- 在战斗中选择卡牌或技能

## 📋 游戏系统说明

### 1. 角色系统
- **5种职业**：战士、法师、盗贼、牧师、弓箭手
- **属性系统**：力量、敏捷、智力、体质、幸运、魅力
- **升级系统**：经验升级，属性加点，技能学习

### 2. 战斗系统
- **回合制战斗**：策略性战斗体验
- **卡牌系统**：战斗中使用卡牌发动技能
- **技能系统**：每个职业都有独特的职业技能
- **状态效果**：眩晕、中毒、燃烧等

### 3. 卡牌系统
- **卡牌稀有度**：普通、稀有、史诗、传说
- **卡牌类型**：攻击、防御、魔法、辅助
- **卡组构建**：自由组合你的战斗卡组

### 4. 地图系统
- **6个地点**：村庄、酒馆、森林、洞穴、山脉、遗迹
- **随机遭遇**：探索时可能遇到怪物
- **地点功能**：休息、交易、接取任务

### 5. 酒馆系统
- **NPC互动**：与酒馆老板、商人等对话
- **商店系统**：购买装备和物品
- **任务板**：接取新的冒险任务
- **休息系统**：恢复生命和魔法

### 6. 任务系统
- **主线任务**：推动剧情发展
- **支线任务**：丰富游戏内容
- **每日任务**：每日更新，获得奖励

### 7. 剧情系统
- **分支剧情**：你的选择影响故事走向
- **多结局**：不同选择导致不同结局
- **AI生成**：动态生成对话和场景描述

### 8. AI系统（可选功能）
- **文本生成**：AI生成对话和描述
- **图像生成**：生成角色头像和场景图片
- **语音合成**：为对话配音

## 🔧 开发模式

### 调试命令（在浏览器控制台中使用）

```javascript
// 添加资源
Game.addGold(1000);        // 添加1000金币
Game.addExp(500);          // 添加500经验

// 完全恢复
Game.fullRestore();        // 完全恢复

// 立即升级
Game.levelUp();            // 立即升级

// 开始战斗
Game.startBattle('slime'); // 开始战斗

// 查看数据
Game.showData();           // 显示所有数据

// 清除存档
Game.clearSave();          // 清除存档

// 获得所有卡牌
Game.getAllCards();        // 获得所有卡牌
```
