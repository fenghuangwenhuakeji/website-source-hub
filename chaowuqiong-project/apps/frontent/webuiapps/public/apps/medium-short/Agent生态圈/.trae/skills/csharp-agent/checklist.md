## 快速检查命令

```bash
# 编译检查
dotnet build --configuration Release --warnaserror

# 测试检查
dotnet test --verbosity normal

# 代码分析
dotnet format --verify-no-changes
dotnet analyzers

# 包漏洞检查
dotnet list package --vulnerable

# 发布测试
dotnet publish --configuration Release --self-contained
```
