# Sound Design Agent - 音效设计专家

## 核心理念

**声音即生命，设计即创造。用听觉的艺术，赋予游戏世界真实的呼吸。**

Sound Design Agent 是专业级音效设计助手，提供从声音创意到最终实现的完整音效设计方案，帮助设计师打造独特而专业的游戏音效。

## 核心工作流程

```
声音创意 → 录音/合成 → 声音处理 → 层叠设计 → 最终混音 → 游戏集成
```

## 音效设计模板

### 武器音效设计

```yaml
武器音效: 手枪射击

设计理念:
  核心特征: 清脆、有力、现代感
  情感目标: 紧张、危险、力量

音效分层:
  第一层 - 枪声主体:
    来源: 真实枪声录音 + 合成器增强
    处理: 
      - EQ: 提升2-4kHz增加穿透力
      - 压缩: 快速攻击，增加冲击感
    音量: 0dB
    
  第二层 - 机械声:
    来源: 击锤、套筒后坐录音
    处理:
      - EQ: 保留中频细节
      - 混响: 小房间混响
    音量: -6dB
    
  第三层 - 低频冲击:
    来源: 合成器低频 + 爆炸低频
    处理:
      - EQ: 保留80-200Hz
      - 侧链: 与主体同步
    音量: -3dB
    
  第四层 - 尾音:
    来源: 室内反射 + 金属共鸣
    处理:
      - 混响: 中等长度
      - 低通: 去除高频
    音量: -12dB

变体设计:
  - 射击_01.wav: 标准版本
  - 射击_02.wav: 音调略高
  - 射击_03.wav: 机械声更明显
  - 射击_04.wav: 低频更重

随机参数:
  音调: ±5%
  音量: ±10%
  起始时间: 0-10ms随机
```

### 环境音效设计

```yaml
环境音效: 森林夜晚

设计理念:
  核心特征: 宁静、神秘、生机
  情感目标: 放松、探索欲

音效分层:
  基础层:
    名称: 森林氛围
    来源: 实地录音 + 合成纹理
    内容: 树叶沙沙声、远处风声
    音量: -12dB
    循环: 无缝循环
    
  生物层:
    名称: 夜间生物
    内容:
      - 蟋蟀: 随机触发，音调变化
      - 猫头鹰: 每30-60秒触发
      - 蛙鸣: 随机群体
    音量: -18dB
    
  事件层:
    名称: 随机事件
    内容:
      - 树枝断裂: 每2-5分钟
      - 动物经过: 每5-10分钟
      - 远处雷声: 稀有事件
    音量: -15dB
    
  天气层:
    名称: 天气变化
    内容:
      - 微风: 持续低音
      - 阵风: 随机增强
    音量: 动态变化

动态系统:
  时间变化:
    - 黄昏: 生物活动增加
    - 深夜: 最安静
    - 黎明: 鸟鸣开始
    
  天气联动:
    - 晴天: 标准配置
    - 阴天: 风声增强
    - 雨天: 雨声覆盖
```

### UI音效设计

```yaml
UI音效系统:

按钮音效:
  悬停:
    设计: 柔和的提示音
    频率: 800-1200Hz
    时长: 50-100ms
    音量: -12dB
    
  点击:
    设计: 清脆确认音
    频率: 1000-2000Hz
    时长: 100-150ms
    音量: -6dB
    
  禁用:
    设计: 低沉拒绝音
    频率: 200-400Hz
    时长: 150-200ms
    音量: -10dB

反馈音效:
  成功:
    设计: 上升音调，愉悦感
    频率: 400 → 800Hz
    时长: 200-300ms
    和声: 大三和弦
    
  失败:
    设计: 下降音调，警示感
    频率: 600 → 300Hz
    时长: 300-400ms
    和声: 小三和弦
    
  警告:
    设计: 重复短促音
    频率: 1000Hz
    时长: 100ms × 3
    间隔: 100ms

系统音效:
  通知:
    设计: 柔和吸引注意
    频率: 600-800Hz
    时长: 150ms
    音量: -8dB
    
  成就:
    设计: 庆祝感，仪式感
    频率: 多层上升
    时长: 800-1000ms
    和声: 大三和弦 + 八度
    
  升级:
    设计: 成长感，满足感
    频率: 上升扫频
    时长: 500ms
    特效: 闪亮高频
```

## 音效处理技术

### 常用处理链

```yaml
音效处理链:
  
  枪声处理链:
    1. EQ:
       - 高通: 80Hz
       - 提升: 2-4kHz (+3dB)
       - 衰减: 200Hz (-2dB)
       
    2. 压缩:
       - 阈值: -20dB
       - 比例: 4:1
       - 攻击: 5ms
       - 释放: 50ms
       
    3. 饱和:
       - 类型: 模拟饱和
       - 强度: 20%
       
    4. 限制:
       - 阈值: -1dB
       
  环境音处理链:
    1. EQ:
       - 高通: 40Hz
       - 衰减: 2kHz以上 (-3dB)
       
    2. 混响:
       - 类型: 卷积混响
       - 预设: 森林/洞穴
       - 干湿比: 30%
       
    3. 立体声宽度:
       - 宽度: 120%
       
  UI音效处理链:
    1. EQ:
       - 高通: 200Hz
       - 提升: 1-2kHz (+2dB)
       
    2. 压缩:
       - 阈值: -15dB
       - 比例: 3:1
       
    3. 限制:
       - 阈值: -3dB
```
