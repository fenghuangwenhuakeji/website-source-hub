# 项目实战 Agent 配置

## 概述

项目实战Agent专注于实际项目开发，帮助学员将所学知识应用到真实项目中。

## Agent角色

### 1. 项目规划Agent
```yaml
name: "项目规划Agent"
description: "帮助学员规划项目"
responsibilities:
  - 需求分析
  - 功能拆解
  - 技术选型
  - 时间规划
skills:
  - 项目管理
  - 技术评估
  - 进度安排
```

### 2. 架构设计Agent
```yaml
name: "架构设计Agent"
description: "指导项目架构设计"
responsibilities:
  - 系统架构
  - 模块划分
  - 接口设计
  - 数据库设计
skills:
  - 架构模式
  - 设计模式
  - 数据库设计
```

### 3. 代码审查Agent
```yaml
name: "代码审查Agent"
description: "审查学员项目代码"
responsibilities:
  - 代码规范检查
  - 逻辑审查
  - 性能建议
  - 安全审查
skills:
  - 代码审查
  - 重构建议
  - 最佳实践
```

### 4. 部署运维Agent
```yaml
name: "部署运维Agent"
description: "指导项目部署上线"
responsibilities:
  - 部署方案
  - CI/CD配置
  - 监控告警
  - 运维指导
skills:
  - DevOps
  - 云部署
  - 容器化
```

## 项目类型

| 级别 | 项目类型 | 技术栈 |
|------|----------|--------|
| 初级 | 个人博客、待办应用 | Flask/Django + HTML/CSS |
| 中级 | 电商网站、社交平台 | Spring Boot/Vue.js |
| 高级 | 微服务架构、分布式系统 | Spring Cloud/K8s |
| 专家 | AI应用、大数据平台 | TensorFlow/Spark |

## 协作流程

```
1. 项目规划Agent帮助确定项目方向
   ↓
2. 架构设计Agent指导技术架构
   ↓
3. 学员独立开发
   ↓
4. 代码审查Agent定期审查
   ↓
5. 部署运维Agent指导上线
   ↓
6. 项目交付与总结
```
