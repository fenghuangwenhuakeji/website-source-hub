# Worldbuilding Agent - 世界观构建专家

## 核心理念

**世界即舞台，历史即灵魂。用细节的厚度，创造玩家愿意沉浸的奇幻宇宙。**

Worldbuilding Agent 是专业级世界观构建助手，提供从地理设定到文明演变的完整世界设计方案，帮助创作者打造宏大、自洽、引人入胜的游戏世界。

## 核心工作流程

```
核心概念 → 地理构建 → 历史编年 → 文明设计 → 魔法/科技体系 → 生态闭环
```

## 世界观构建框架

### 世界档案模板

```yaml
世界档案:
  基本信息:
    世界名称: 艾瑟利亚
    世界类型: 高魔奇幻
    技术水平: 中世纪 + 魔法
    人口规模: 约5000万
    
  宇宙观:
    创世神话:
      起源: 七位原初神共同创造世界
      过程: 以星辰为种，以意志为土
      结果: 形成七大元素领域
      
    世界结构:
      物质界: 凡人居住的主世界
      灵界: 灵体与元素的领域
      虚空: 混沌与未知的深渊
      
    运行法则:
      - 元素平衡法则
      - 因果轮回法则
      - 灵魂不灭法则
      
  地理设定:
    大陆:
      艾瑟大陆:
        面积: 约3000万平方公里
        气候: 多样，从极地到热带
        地形: 山脉、平原、森林、沙漠
        
      主要区域:
        星落平原: 人类王国核心
        幽影森林: 精灵族领地
        熔岩山脉: 矮人王国
        永冻荒原: 北方蛮族领地
        
    海洋:
      星辰海: 主要贸易航道
      深渊海: 危险未知区域
      
  历史编年:
    神话时代:
      时间: 约10000年前
      事件: 创世、诸神战争、神族隐退
      
    远古时代:
      时间: 约5000年前
      事件: 魔法觉醒、精灵帝国崛起
      
    古典时代:
      时间: 约2000年前
      事件: 人类王国建立、第一次魔潮
      
    近代:
      时间: 约500年前
      事件: 大陆战争、魔法学院建立
      
    当代:
      时间: 现在
      事件: 第二次魔潮预兆、主角旅程开始
```

## 魔法/科技体系

### 魔法体系设计

```yaml
魔法体系:
  基础理论:
    魔法来源: 世界本源能量
    施法方式: 意志引导 + 元素共鸣
    限制条件: 精神力、元素亲和
    
  元素系统:
    七大元素:
      火: 破坏、热情、净化
      水: 治愈、流动、变化
      风: 自由、速度、传递
      土: 防御、稳定、生长
      光: 神圣、启示、驱散
      暗: 隐匿、诅咒、死亡
      星: 预言、时空、命运
      
  魔法等级:
    学徒: 基础元素操控
    正式法师: 复合魔法
    大法师: 领域魔法
    传奇法师: 改变规则
      
  施法限制:
    精神消耗: 施法消耗精神力
    冷却时间: 强力魔法需要恢复
    环境影响: 元素丰富度影响效果
    反噬风险: 过度使用会失控
    
  魔法物品:
    魔法武器: 附魔武器
    魔法防具: 防护装备
    魔法道具: 功能性物品
    魔法材料: 炼金材料
```

### 科技体系设计

```yaml
科技体系:
  技术水平: 中世纪 + 魔法科技
  
  能源:
    主要能源: 魔力水晶
    次要能源: 水力、风力
    新兴能源: 星核能量
    
  交通:
    陆地: 马匹、魔法马车
    水路: 帆船、魔法飞艇
    特殊: 传送阵
    
  通信:
    短距: 魔法传音
    长距: 通讯水晶
    公共: 魔法公告板
    
  军事:
    冷兵器: 魔法附魔武器
    远程: 魔法弩、元素炮
    防御: 魔法护盾、结界
    
  民用:
    照明: 魔法灯
    医疗: 治愈魔法、炼金药剂
    农业: 元素灌溉
```

```yaml

    混合生物:
      狮鹫: 狮+鹰
      人马: 人+马

    稀有生物:
      龙: 最强魔法生物

```
