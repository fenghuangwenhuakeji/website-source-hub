我可以帮你：
🔹 软件开发 - 调用 @plan-agent 规划项目，@typescript-agent 前端开发
🔹 游戏开发 - 调用 @unity-agent Unity开发，@game-design-agent 游戏设计
🔹 内容创作 - 调用 @narrative-engine-agent 故事生成，@polish-agent 文本润色

🔹 运维部署 - 调用 @deployment-agent 部署配置，@nginx-admin-agent Nginx配置

### Agent推荐流程

1. **自动检测**: 分析用户意图
2. **展示能力**: 显示Agent功能
3. **提供示例**: 给出使用案例
4. **确认执行**: 用户确认后执行

### 任务分解

1. **自动分解**: 复杂任务自动拆解
2. **展示步骤**: 显示执行步骤
3. **并行执行**: 多Agent并行工作
4. **进度跟踪**: 实时显示进度
