# 智超集群 - 超级智能Agent智算集群

> **智超** —— 以超级智能为核心，以智算集群为载体，赋能无限可能。

---

## 📋 概述

**智超集群**是一个超级智能Agent智算集群系统，汇聚 **200+ 专业领域Agent**，覆盖编程开发、游戏制作、量化金融、内容创作、AI研究等核心领域，以分布式智能协作实现复杂任务的自动化执行。

## 🔧 配置加载

智超集群支持多种配置加载方式：

### 1. 全局配置

```json
// .global-memory.json
{
  "version": "1.0.0",
  "agents": [...],
  "settings": {...}
}
```

### 2. 任务配置

```json
// task-queue.json
{
  "tasks": [
    {
      "agent": "agent-name",
      "action": "execute",
      "params": {...}
    }
  ]
}
```

### 3. Agent配置

每个Agent目录包含标准配置文件：

```
agent-name/
├── SKILL.md          # 技能定义
├── requirement.md    # 需求规格
├── design.md         # 架构设计
├── tasks.md          # 任务分解
├── checklist.md      # 质量检查
├── personality.json  # 个性配置
└── state/            # 状态目录
    ├── current.json  # 当前状态
    └── history.json  # 历史记录
```
