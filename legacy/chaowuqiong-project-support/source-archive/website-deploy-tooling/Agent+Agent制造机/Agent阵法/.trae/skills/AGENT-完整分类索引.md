## 🔥 常用工作流推荐

### 📚 小说创作工作流
```
1. @brainstorm-agent     → 生成创意概念和标题
2. @outline-agent        → 创建详细大纲
3. @worldbuilding-agent  → 构建世界观
4. @character-design-agent → 设计角色
5. @narrative-engine-agent → 撰写正文
6. @polish-agent         → 润色优化
```

### 💻 编程开发工作流
```
1. @language-selector-agent → 技术选型
2. @plan-agent           → 项目规划
3. @spec-agent           → 需求规格
4. @[语言Agent]          → 代码实现
5. @debug-agent          → 测试优化
```

### 🎮 游戏开发工作流
```
1. @game-design-agent    → 游戏设计
2. @unity-agent/@godot-* → 引擎开发
3. @level-design-agent   → 关卡设计
4. @shader-agent/@vfx-agent → 视觉特效
5. @game-audio-agent     → 音频制作
6. @game-testing-agent   → 测试验证
```

### 🎬 剧本创作工作流
```
1. @three-act-agent      → 三幕式结构
2. @character-arc-agent  → 角色弧线
3. @dialogue-writer-agent → 对话写作
4. @conflict-agent       → 冲突设计
5. @screenplay-format-agent → 格式规范
```

### 🎨 分镜制作工作流
```
1. @script-creator-agent → 剧本创作
2. @storyboard-agent     → 故事板设计
3. @nanobanana-grid-agent → AI图像提示
4. @comic-creator-agent  → 漫画制作
```

### 📈 量化金融工作流
```
1. @data-acquisition-agent → 数据采集
2. @data-processing-agent → 数据处理
3. @strategy-development-agent → 策略开发
4. @backtesting-agent    → 回测验证

```

### 方式1: 直接引用
```

```

```
先用 @brainstorm-agent 生成创意，
再用 @outline-agent 创建大纲，
```
