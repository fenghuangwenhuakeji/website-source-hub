核心原则：一切镜头语言服务于“权力”的展示与更迭。主角陈默是权力的绝对中心。

第一法则：镜头运动与角色权力等级绑定

1.  【权力赋予镜头】
    ◦   功能： 用于宣告主角的成长、胜利或绝对掌控。

    ◦   规范：

        ▪   运镜方式： 低角度仰拍 (Low-Angle Shot) + 平稳的推近 (Dolly In)。

        ▪   景别： 从中景缓慢推至近景或特写。

        ▪   适用情境： 主角获得关键能力、战胜强大敌人后。

        ▪   AI指令示例： [权力赋予镜头]：陈默获得系统奖励后，使用低角度仰拍，并平稳推近至其面部特写，捕捉其掌控一切的微笑。

2.  【压迫与窥视镜头】
    ◦   功能： 制造恐惧、不确定感和被动感。

    ◦   规范：

        ▪   运镜方式： 手持晃动 (Handheld) + 框景构图 (Framing Shot)（如通过门缝、窗户）。

        ▪   景别： 特写或极近景，聚焦于角色惊恐的眼睛或局促的动作。

        ▪   适用情境： 战斗、未知危险逼近、角色处于弱势时。

        ▪   AI指令示例： [压迫与窥视镜头]：对战剧情 使用手持晃动镜头，模拟陈默的主观视角 
3.  【评估与审视镜头】
    ◦   功能： 表现角色（尤其是陈默）冷静分析、评估局势或他人价值。

    ◦   规范：

        ▪   运镜方式： 缓慢的横移 (Slow Pan) 或 静止固定镜头 (Static Shot)。

        ▪   景别： 中景到全景。

        ▪   焦点： 镜头模拟陈默的视线，扫过目标（如一件物品、一个潜在队友）。

        ▪   适用情境： 陈默判断新来者、评估战利品、制定战术时。

        ▪   AI指令示例： [评估与审视镜头]：其他角色与主角对话 ，使用缓慢的横移镜头，从她的脸扫到她和身上，胸前，腿上，再回到陈默的脸上，表现其正在权衡。

第二法则：景别与角色关系及情绪绑定

1.  【亲密/威胁镜头】
    ◦   功能： 极致化情色张力或致命威胁。

    ◦   规范：

        ▪   景别： 大特写 (Extreme Close-Up)。

        ▪   焦点： 非完整的脸，而是局部。如：颤抖的嘴唇、滚动的喉结、闪烁的眼神、女性角色颤动的胸部，修长的长腿等细节表现 
        ▪   适用情境：暧昧关系  调情、致命一击前、极度恐惧时。

        ▪   AI指令示例： [亲密/威胁镜头]：女角色贴近陈默边时，给女角色胸部贴近陈默，首部抚摸陈默，给陈默滚动的喉结一个大特写，表现其生理反应与内心的克制。

2.  【权力关系镜头】
    ◦   功能： 清晰展示角色在空间中的等级地位。

    ◦   规范：

        ▪   构图： 必须让陈默处于画面的视觉中心或前景主导位置。

        ▪   他人位置： 其他角色呈环绕、依附或处于背景/边缘。

        ▪   适用情境： 团队讨论、多人场景。

        ▪   AI指令示例： [权力关系镜头]：在场景，陈默在主位，其他女性成员以他为中心呈环绕状构图，凸显其主导地位。

第三法则：剪辑节奏与叙事节奏绑定

1.  【混乱爆发节奏】
    ◦   功能： 表现突发战斗、混乱和恐惧。

    ◦   规范： 快速剪辑 (Fast-Paced Editing)，镜头长度短，配合手持晃动。

    ◦   AI指令示例： [混乱爆发节奏]：对战情景，使用一系列少于2秒的快速晃动镜头，表现瞬间的混乱与惊恐。

2.  【掌控从容节奏】
    ◦   功能： 表现陈默掌控局面的安全感和强大。

    ◦   规范： 长镜头 (Long Take) 或 缓慢剪辑 (Slow-Paced Editing)，镜头长度显著变长。

    ◦   AI指令示例： [掌控从容节奏]：陈默在场景内观察外界时，使用一个长达10秒的固定长镜头，表现其从容。
当生成任何一场戏的分镜时，先判断该场戏的核心叙事目标（例如：是“展示权力”还是“制造压迫”？是“情色调情”还是“评估价值”？），然后从上述法则中调用对应的镜头类型进行组合与构建。确保每一格画面都遵循“权力视觉化”的最高原则。

第四法则：材质与细节真实可信（或风格化一致）
- 材质参数合理，反射、粗糙度、置换等符合物理或风格设定。
- 细节（灰尘、划痕、磨损）提升真实感。
- 不同物体之间的材质对比要自然。

第五法则：特效服务画面，不喧宾夺主
- 特效（粒子、流体、烟雾、爆炸）与场景融合。
- 特效节奏与镜头节奏匹配。
- 不滥用特效掩盖叙事或技术不足。

第六法则：色彩与氛围统一
- 色彩风格明确（冷色、暖色、高饱和、低饱和等）。
- 色彩用于表达情绪或区分场景。
- 避免色彩杂乱导致视觉疲劳。

第七法则：画面的品质要求
电影级别的光影效果，人物和场景完美通融，光影完美，大师级作品
