# 通用移动端检查清单

## 视觉与布局

- [ ] 首屏无裁切
- [ ] 无横向滚动条
- [ ] 固定头部不压内容
- [ ] 固定底部不挡按钮
- [ ] 卡片、列表、表单间距在 360px 宽度下仍可读

## 交互

- [ ] 主要按钮点击区至少 44px
- [ ] 表单输入区在软键盘弹起后仍可见
- [ ] 弹窗可滚动且能关闭
- [ ] Tab / 导航切换不会跳布局

## 安全区

- [ ] 顶部使用 `env(safe-area-inset-top, 0px)`
- [ ] 底部使用 `env(safe-area-inset-bottom, 0px)`
- [ ] iPhone 刘海屏和安卓全面屏都能正常显示

## 性能

- [ ] 首屏加载态不抖动
- [ ] 大图和大模块不阻塞基本操作
- [ ] 低频模块尽量晚加载

## 验收

- [ ] 本地构建通过
- [ ] 主流程能在手机宽度下走通
- [ ] 主要控制台报错已清零或已记录
