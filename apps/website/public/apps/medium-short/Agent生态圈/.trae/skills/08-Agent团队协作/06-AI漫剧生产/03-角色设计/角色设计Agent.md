# 角色设计 Agent 配置

## 概述

角色设计Agent专注于角色视觉形象的设计和细化。

## Agent角色

### 1. 角色原画Agent
```yaml
name: "角色原画Agent"
description: "设计角色原画"
responsibilities:
  - 角色概念
  - 造型 design
  - 服装设计
  - 配色方案
skills:
  - 角色设计
  - 人体结构
  - 色彩理论
```

### 2. 人设细化Agent
```yaml
name: "人设细化Agent"
description: "细化角色设定"
responsibilities:
  - 三视图
  - 细节设计
  - 材质表现
  - 比例规范
skills:
  - 细节刻画
  - 规范制定
  - 多角度
```

### 3. 表情动作Agent
```yaml
name: "表情动作Agent"
description: "设计表情和动作"
responsibilities:
  - 表情 sheet
  - 动作姿态
  - 性格表现
  - 动态线
skills:
  - 表情设计
  - 动作设计
  - 动态表现
```

## 协作流程

```
1. 角色原画Agent设计概念
   ↓
2. 人设细化Agent细化设定
   ↓
3. 表情动作Agent设计动态
   ↓
4. 角色设定完成
```
