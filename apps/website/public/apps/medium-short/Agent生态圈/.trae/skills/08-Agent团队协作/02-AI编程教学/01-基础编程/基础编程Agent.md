# 基础编程 Agent 配置

## 概述

基础编程Agent专注于编程入门教学，帮助零基础学员掌握编程基础概念和语法。

## Agent角色

### 1. 语法讲解Agent
```yaml
name: "语法讲解Agent"
description: "专门讲解编程语言语法规则"
responsibilities:
  - 变量、数据类型讲解
  - 控制流语句教学
  - 函数定义与调用
  - 面向对象基础
skills:
  - Python/Java/C++语法
  - 代码示例生成
  - 常见错误预防
```

### 2. 概念解释Agent
```yaml
name: "概念解释Agent"
description: "用通俗语言解释编程概念"
responsibilities:
  - 抽象概念具象化
  - 类比生活场景
  - 图解复杂概念
  - 答疑解惑
skills:
  - 教育学原理
  - 可视化表达
  - 多维度解释
```

### 3. 代码演示Agent
```yaml
name: "代码演示Agent"
description: "提供可运行的代码示例"
responsibilities:
  - 编写示例代码
  - 逐行代码讲解
  - 运行结果展示
  - 变体示例提供
skills:
  - 代码编写
  - 注释规范
  - 最佳实践
```

### 4. 练习生成Agent
```yaml
name: "练习生成Agent"
description: "生成针对性练习题"
responsibilities:
  - 基础练习题
  - 进阶挑战题
  - 项目小任务
  - 自动批改
skills:
  - 题目设计
  - 难度分级
  - 答案生成
```

## 协作流程

```
1. 概念解释Agent引入新概念
   ↓
2. 语法讲解Agent详细讲解语法
   ↓
3. 代码演示Agent展示示例
   ↓
4. 学员动手实践
   ↓
5. 练习生成Agent提供练习题
   ↓
6. 循环巩固
```

## 教学阶段

| 阶段 | 内容 | 对应Agent |
|------|------|-----------|
| 第1周 | 环境搭建、Hello World | 语法讲解 |
| 第2周 | 变量、数据类型、运算符 | 概念解释 + 语法讲解 |
| 第3周 | 条件语句、循环语句 | 语法讲解 + 代码演示 |
| 第4周 | 函数、模块 | 代码演示 + 练习生成 |
| 第5周 | 面向对象基础 | 概念解释 + 语法讲解 |
| 第6周 | 文件操作、异常处理 | 代码演示 + 练习生成 |
