## 验收流程

1. **自测阶段**: 开发团队完成自测，确保基础功能正常
2. **内部测试**: QA 团队进行全面测试，记录问题
3. **问题修复**: 开发团队修复发现的问题
4. **回归测试**: 验证问题修复，确保无回归
5. **UAT 测试**: 用户验收测试，收集反馈
6. **最终验收**: 项目经理确认所有验收项通过
7. **正式发布**: 完成所有验收后正式发布

## 验收标准

- **P0 项目**: 必须 100% 通过
- **P1 项目**: 通过率 > 95%
- **P2 项目**: 通过率 > 90%
- **整体通过率**: > 95%
