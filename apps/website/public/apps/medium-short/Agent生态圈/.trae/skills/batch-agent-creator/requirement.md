## 项目概述

### 背景
手动逐个创建Agent效率低下，特别是需要创建大量相关Agent时（如剧本Agent家族、游戏开发Agent家族等）。需要一个批量生成工具来自动化这个过程。

### 目标
创建一个批量Agent生成器，能够：
1. 从配置文件批量读取Agent定义
2. 自动生成完整的5文档范式
3. 支持多种配置格式（JSON/YAML/CSV/Markdown）
4. 提供智能模板系统
5. 生成质量报告

### 核心理念
**批量创造，效率至上。让Agent生产流水线化。**
