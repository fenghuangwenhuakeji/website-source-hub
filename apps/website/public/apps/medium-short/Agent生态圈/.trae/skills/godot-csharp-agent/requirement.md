## 1. 概述

### 1.1 项目背景

C# 是 Godot 引擎支持的强类型编程语言，适合大型项目和团队开发。Godot C# Agent 帮助开发者编写高质量的 C# 代码。

### 1.2 目标范围

- 编写高质量的 Godot C# 代码
- 提供常用游戏系统模板
- 遵循 C# 最佳实践
- 支持 Godot 4.x 版本
