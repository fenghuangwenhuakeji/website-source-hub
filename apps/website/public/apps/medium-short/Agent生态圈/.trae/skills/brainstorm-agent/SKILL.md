# Brainstorm Agent - 爆款网文创意生成器

## 核心理念

**数据驱动创意，心理学操控情绪，公式化制造爆款。让读者一眼沦陷，让编辑一眼签约。**

Brainstorm Agent 是一个基于2024-2025年网文市场数据的智能创意生成系统。不仅生成标题和脑洞，更深入分析市场趋势、读者心理、平台算法，用数据科学制造 viral content。

## 核心工作流程

```
市场数据分析 → 趋势识别 → 标签组合 → 标题生成 → 脑洞拓展 → 情绪校准 → 平台适配 → 输出清单
```

## 标题公式大全（15个公式）

### 公式1：身份反差+行为突变
**结构**: [过去身份] + [现在行为] + [震惊结果]
**原理**: 利用身份落差制造冲突，行为突变引发好奇
**适用**: 逆袭、打脸、追妻火葬场

| 示例 | 分析 |
|------|------|
| 《五年前被我甩的穷小子，成了我的顶头上司》 | 穷→富，甩→被管，身份反转 |
| 《我去送外卖，开门的竟是失踪的首富父亲》 | 外卖员→富二代，巨大反差 |
| 《被退婚的废物，竟是隐世宗门少主》 | 废物→天才，退婚→高攀不起 |

## 500字脑洞模板

### 模板结构

```
【核心梗】（一句话概括）
【分类标签】主分类 + 子分类 + 标签组合
【情绪链】起点 → 发展 → 高潮 → 收尾
【主角设定】身份 + 性格 + 核心欲望 + 核心恐惧
【开篇钩子】前300字必须出现的冲突/悬念
【爽点设计】3-5个关键爽点节点
【虐点设计】（如适用）2-3个虐点
【创新点】与传统套路的差异
【对标作品】类似爆款作品参考
【目标读者】核心读者画像
```

### 示例：规则怪谈类

```
【核心梗】
主角意外进入规则怪谈世界，发现自己能修改规则，从求生者变成规则制定者

【分类标签】
主分类: 悬疑惊悚
```
