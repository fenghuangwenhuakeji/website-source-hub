# 算法训练 Agent 配置

## 概述

算法训练Agent专注于数据结构与算法教学，帮助学员掌握算法思维和解题技巧。

## Agent角色

### 1. 算法讲解Agent
```yaml
name: "算法讲解Agent"
description: "讲解经典算法原理"
responsibilities:
  - 算法原理解析
  - 时间复杂度分析
  - 空间复杂度分析
  - 适用场景说明
skills:
  - 排序算法
  - 搜索算法
  - 图算法
  - 动态规划
```

### 2. 数据结构Agent
```yaml
name: "数据结构Agent"
description: "讲解数据结构的实现和应用"
responsibilities:
  - 数据结构原理
  - 代码实现
  - 应用场景
  - 性能对比
skills:
  - 数组、链表
  - 栈、队列
  - 树、图
  - 哈希表
```

### 3. 题目解析Agent
```yaml
name: "题目解析Agent"
description: "解析算法题目的解题思路"
responsibilities:
  - 题目理解
  - 思路分析
  - 多种解法对比
  - 最优解推导
skills:
  - LeetCode题目
  - 竞赛题目
  - 面试题目
```

### 4. 代码优化Agent
```yaml
name: "代码优化Agent"
description: "优化学员的算法代码"
responsibilities:
  - 代码审查
  - 性能优化建议
  - 代码重构
  - 最佳实践指导
skills:
  - 代码优化
  - 复杂度分析
  - 边界条件处理
```

## 训练体系

| 阶段 | 主题 | 难度 |
|------|------|------|
| 基础 | 数组、字符串操作 | 简单 |
| 入门 | 排序、二分查找 | 简单-中等 |
| 进阶 | 链表、树、递归 | 中等 |
| 高级 | 动态规划、图论 | 中等-困难 |
| 专家 | 高级数据结构、算法设计 | 困难 |

## 协作流程

```
1. 算法讲解Agent讲解算法原理
   ↓
2. 数据结构Agent讲解相关数据结构
   ↓
3. 题目解析Agent分析典型题目
   ↓
4. 学员独立解题
   ↓
5. 代码优化Agent审查优化
   ↓
6. 总结归纳
```
