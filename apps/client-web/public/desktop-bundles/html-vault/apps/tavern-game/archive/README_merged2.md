# 传奇酒馆 - 完整角色扮演游戏

一个功能丰富的网页角色扮演游戏，包含完整的游戏系统。

## 🎮 游戏特性

### 核心系统
- **角色系统** - 创建自定义角色，5种职业可选
- **升级系统** - 经验升级，属性加点，技能学习
- **职业系统** - 战士、法师、盗贼、牧师、弓箭手
- **战斗系统** - 回合制战斗，技能系统，卡牌系统
- **卡牌系统** - 卡牌收集，卡组构建，战斗使用
- **剧情系统** - 分章节剧情，分支选择
- **任务系统** - 主线任务，支线任务，每日任务
- **地图系统** - 探索多个地点，随机遭遇
- **酒馆系统** - NPC对话，商店，住宿，任务板
- **保存系统** - 多存档位，自动保存，导入导出

### 游戏内容
- **5种职业** - 各具特色的战斗风格
- **15+技能** - 职业专属技能
- **20+卡牌** - 攻击、防御、魔法、辅助
- **10+怪物** - 从史莱姆到巨龙
- **6个地点** - 村庄、酒馆、森林、洞穴、山脉、遗迹
- **多个NPC** - 丰富的对话和互动
- **道具装备** - 武器、护甲、消耗品

## 🚀 快速开始

### 1. 打开游戏
直接用浏览器打开 `index.html` 文件即可开始游戏。

### 2. 创建角色
- 输入角色名称
- 选择职业（战士/法师/盗贼/牧师/弓箭手）

### 3. 开始冒险
- 完成序章剧情
- 在酒馆接取任务
- 探索地图，战斗升级
- 收集卡牌，构建卡组
- 转职进阶，学习新技能

## 🎯 操作指南

### 键盘快捷键
- `1` - 地图视图
- `2` - 酒馆视图
- `3` - 角色面板
- `4` - 卡牌收藏
- `5` - 职业系统
- `ESC` - 打开菜单

### 战斗操作
- **攻击** - 普通攻击
- **技能** - 使用职业技能（消耗MP）
- **卡牌** - 打出战斗卡牌（消耗能量）
- **防御** - 减少受到的伤害
- **物品** - 使用消耗品
- **逃跑** - 尝试逃离战斗

### 地图操作
- 点击地点图标进行移动
- 不同地点有不同的功能和怪物

## 🎨 游戏系统详解

### 1. 职业系统
每个职业有独特的属性加成和技能：

- **战士** - 高生命高防御，擅长近战
- **法师** - 高魔法值，远程魔法攻击
- **盗贼** - 高速度高暴击，擅长背刺
- **牧师** - 治疗支援，神圣力量
- **弓箭手** - 远程精准射击，陷阱技能

### 2. 战斗系统
- 回合制战斗
- 技能系统（消耗MP）
- 卡牌系统（消耗能量）
- 暴击、闪避、眩晕等状态效果
- 持续伤害效果

### 3. 卡牌系统
- 卡牌分类：攻击、防御、魔法、辅助
- 卡牌稀有度：普通、稀有、史诗、传说
- 构建战斗卡组
- 卡牌收藏和过滤

### 4. 升级系统
- 经验升级
- 技能点分配
- 属性加点
- 学习新技能

### 5. 任务系统
- 主线任务
- 支线任务
- 每日任务
- 任务奖励

### 6. 地图系统
- 多个可探索地点
- 点击移动
- 随机怪物遭遇
- 地点特殊功能
