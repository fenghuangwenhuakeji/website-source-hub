## 🚀 如何使用

### 启动游戏
```bash
# 方法1: 直接打开
双击 UNIFIED_TAVERN_GAME/index.html

# 方法2: 使用本地服务器 (推荐)
python -m http.server 8000
# 然后访问 http://localhost:8000
```

### 游戏操作
1. 点击"新游戏"开始
2. 输入角色名称
3. 选择职业 (战士/法师/盗贼/牧师/弓箭手)
4. 探索地图、战斗、完成任务
