RNN 序列处理

RNN 在每个时间步接收输入，并结合上一时刻的隐藏状态。这使得它能处理变长序列。

Transformer 注意力机制

注意力权重决定每个位置对其他位置的关注程度。高亮连线越粗 = 注意力越强。
