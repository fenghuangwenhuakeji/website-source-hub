## 三、核心模块说明

### 3.1 领域层 (Domain)

| 领域 | 主要实体 | 说明 |
|------|----------|------|
| **User** | User, UserProfile, AgeGroup | 用户管理、年龄分组、家长账户 |
| **Constellation** | Sign, Horoscope, Compatibility | 12星座、运势、配对分析 |
| **Psychology** | Test, Question, TestResult, PersonalityType | 心理测试、MBTI/NPTI等 |
| **Divination** | TarotCard, TarotSpread, BaziChart, Wuxing, AstroChart | 塔罗、八字、五行、星盘 |
| **Social** | Post, Comment, Follow | 社区帖子、评论、关注 |
| **Safety** | SafetyRule, BlockList, CrisisProtocol | 内容安全、黑名单、危机干预 |
| **Content** | DailyQuote, Article | 每日签语、文章内容 |

### 3.2 服务层 (Services)

| 服务 | 职责 |
|------|------|
| **HoroscopeService** | 运势计算、配对分析、幸运元素 |
| **TestService** | 测试题库管理、评分引擎、报告生成 |
| **TarotService** | 洗牌、抽牌、牌阵、解读 |
| **BaziService** | 八字计算、五行分析、命理报告 |
| **ChatService** | AI对话、会话管理 |
| **CommunityService** | 帖子、评论、点赞 |
| **UserService** | 用户CRUD、权限管理 |
| **ContentService** | 每日签语、文章管理 |
| **SafetyService** | 敏感词过滤、危机检测 |

### 3.3 基础设施层 (Infrastructure)

| 组件 | 职责 |
|------|------|
| **LLMAdapter** | 统一的大模型调用接口，支持多种LLM |
| **StorageService** | 本地存储封装 |
| **CacheService** | 内存缓存，支持TTL |

## 四、使用方式

```javascript
// 初始化应用
import { StarWhispersApp } from './architecture/index.js';

const app = StarWhispersApp.getInstance({
    llm: { provider: 'openai', apiKey: 'xxx' }
});

await app.init();

// 获取运势
const horoscope = app.getHoroscope('白羊座', 'today');

// 塔罗占卜
const reading = app.tarotReading('three', '近期感情如何？');

// AI聊天
const response = await app.chat('你好');

// 八字分析
const bazi = app.baziAnalysis('1990-01-15', '12:30', 'male');
```

## 五、差异化优势

相比测测，星语心伴具有以下独特优势：

1. **全年龄段覆盖** - 支持6岁+用户，三套主题自动切换
2. **AI深度陪伴** - 基于人格+星座的AI角色个性化定制
3. **专业安全机制** - 心理危机识别与干预机制
4. **融合创新** - 心理学(NPTI) + 星座文化 + AI的完美融合
