游戏规则

卡牌Roguelike 地牢探索

基本流程：
每回合获得 3 点能量，从牌堆抽 1 张牌。打出卡牌消耗能量。结束回合后敌人执行意图。

卡牌类型：
攻击牌 — 对敌人造成伤害
防御牌 — 获得护甲，抵挡下次受击
技能牌 — 各种特殊效果

护甲机制：
护甲可以抵消伤害，受到攻击时先扣护甲再扣生命。护甲在你回合开始时清零。

敌人意图：
左侧面板显示敌人下回合的行动：攻击（红）、防御（蓝）、强化（紫），帮助你提前规划。

关键词：
中毒 — 回合开始时损失生命
力量 — 攻击牌额外造成等量伤害
脆弱 — 攻击牌额外造成25%伤害
