# 凤煌引擎开放 API 本地版隐私政策

**版本:** 2026-05-15  
**产品名称:** 凤煌引擎开放 API 本地版  
**软件提供者:** 南京凤煌文化科技有限公司

本政策说明凤煌引擎开放 API 本地版如何处理数据。本版本没有登录注册、会员充值、激活码或公司后端模型代理；默认情况下，公司不通过本版本收集、上传或托管您的作品内容和 API Key。

## 1. 默认本地处理

开放 API 本地版默认在您的浏览器本地运行。以下数据通常保存在本机浏览器 IndexedDB、localStorage、缓存或您主动导出的文件中：

- 项目、卷、章节、大纲、设定、实体、图谱、RAG 上下文、记忆数据。
- 提示词、工具配置、写作偏好、界面设置。
- 您自行配置的 API Base URL、API Key、模型名称和参数。
- 导入的图书、素材、拆书结果、导出文件。

公司默认无法访问这些本地数据，也无法在您清理浏览器数据、硬盘损坏、系统重装或更换浏览器后为您恢复。

## 2. 第三方 API 数据发送

当您主动配置并使用第三方模型 API 时，本软件会从您的浏览器直接向您填写的 API 地址发送请求。发送内容可能包括：

- 您输入的提示词。
- 当前章节、上下文、设定、拆书片段、RAG 片段。
- 模型参数、模型名称、API Key 或鉴权头。
- 第三方模型返回的输出内容。

这些信息由您选择的第三方 API 服务商处理。第三方服务商如何收集、保存、使用、训练、共享或跨境传输数据，以该第三方的服务协议和隐私政策为准。请在配置 API 前自行阅读并判断风险。

## 3. 公司不默认收集的信息

在不接入公司后端服务的情况下，本版本不默认收集：

- 手机号、微信 openid、unionid。
- 会员、充值、订单、支付流水。
- 完整作品库。
- 您配置的第三方 API Key。
- AI 输入输出内容。

如果未来您主动启用公司提供的云同步、商业会员、在线模型代理、客服反馈、错误上报或其他联网服务，应以对应服务的隐私政策或单独告知为准。

## 4. 浏览器与本地安全

您的本地数据安全取决于设备、浏览器、系统账号和扩展环境。建议您：

- 不在公共电脑或不可信设备上保存 API Key。
- 不安装来源不明的浏览器插件。
- 定期导出项目备份。
- 使用系统账号密码、磁盘加密和可信杀毒软件。
- 在共享电脑上使用后清理浏览器数据。

因您设备被他人使用、浏览器插件读取、系统感染恶意软件、API Key 泄露或本地备份管理不当造成的损失，公司在法律允许范围内不承担超出自身过错范围的责任。

## 5. 您的控制权

您可以通过以下方式管理本地数据：

- 在软件内使用导出、备份、清理功能。
- 在浏览器设置中删除 IndexedDB、localStorage、Cookie 和缓存。
- 删除您导出的本地文件。
- 删除或替换 API Key。
- 停止使用某个第三方 API 服务商。

请注意，清理浏览器数据可能导致项目、配置和 API Key 无法恢复。

## 6. 未成年人

未成年人使用本软件应取得监护人同意，并避免向第三方模型服务商发送个人信息、隐私内容、学校信息、家庭住址、联系方式或其他敏感内容。监护人应帮助未成年人理解第三方 API 和 AI 输出风险。

## 7. 政策更新

公司可能根据产品变化、法律法规或分发方式更新本政策。如果开放版未来增加联网服务、错误上报、云同步、账号或商业功能，我们会通过版本说明、页面提示或单独政策说明新的个人信息处理规则。
