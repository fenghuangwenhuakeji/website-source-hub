### 严重程度定义
- **严重** - 影响核心功能，必须修复
- **中等** - 影响非核心功能，建议修复
- **轻微** - 不影响功能，可后续优化

### 验收标准
- **通过**: 通过率 >= 95%，无严重问题
- **有条件通过**: 通过率 >= 90%，无严重问题
- **不通过**: 通过率 < 90%，或存在严重问题
