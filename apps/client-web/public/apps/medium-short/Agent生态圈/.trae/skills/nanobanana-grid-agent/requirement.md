## 1. 概述

### 1.1 项目背景

AI绘画工具如Midjourney、Stable Diffusion等支持生成多面板图像。NanoBananaPro Grid Agent 帮助创作者生成专业的宫格提示词，用于角色展示和设定。

### 1.2 目标范围

- 支持4/6/8/9/16宫格
- 专业镜头语言体系
- 一致性规则保证
- 可直接使用的提示词

### 1.3 核心理念

**一图胜千言，宫格展全貌。用镜头语言讲述角色的完整故事。**

```
角色设定 → 宫格选择 → 镜头分配 → 提示词生成 → 一致性检查 → 输出
```

## 5. 一致性规则

### 5.1 必须保持一致

- 角色外貌（发型、发色、瞳色、身高、体型）
- 服装造型（款式、颜色、配饰、纹理）
- 光影方向（主光源位置、强度、色温）
- 色调风格（整体色彩倾向、饱和度）
- 艺术风格（写实/动漫/油画/像素等）

### 5.2 可以变化

- 姿势动作（站、坐、走、跑等）
- 表情神态（喜、怒、哀、乐等）
- 镜头角度（平、仰、俯、侧等）
- 景别大小（远、中、近、特等）
- 背景环境（室内、室外、抽象等）
