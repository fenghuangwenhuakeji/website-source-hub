## 1. 概述

### 1.1 项目背景

网文市场竞争激烈，一个吸引人的标题和创意概念是作品成功的关键。Brainstorm Agent 旨在帮助创作者快速生成符合市场趋势的爆款标题和详细的故事概念。

### 1.2 目标范围

- 生成100+个网文标题
- 每个标题配500字脑洞
- 覆盖18个主分类
- 融入最新网络热点
- 符合当下读者口味

### 1.3 核心理念

**不够新、不够给力、不够爽？上点实力！反转反转再反转，让人颅内高潮！**

```
市场分析 → 标签组合 → 标题生成 → 脑洞拓展 → 输出清单
```

## 4. 标题公式

### 4.1 公式1：身份反差+行为突变

**结构**: 过去身份 → 现在身份 → 冲突

**示例**:
- 《五年前被我甩的穷小子，成了我的顶头上司》
- 《我去送外卖，开门的竟是失踪的首富父亲》

### 4.2 公式2：平淡开局+意外转折

**结构**: 日常场景 → 悬念/转折

**示例**:
- 《她说给我"惊喜"，真相却是……》
- 《豪门阔太，你欠我的拿什么还》

### 4.3 公式3：暧昧情景+合理借口

**结构**: 暧昧场景 → 反转解释

**示例**:
- 《他把我堵在墙角，只是为了帮我拿掉头上的苍蝇》
- 《约定好禁欲一周，奈何美女校花老婆让我把持不住》

## 5. 脑洞模板

### 5.1 500字脑洞结构

```
1. 核心梗（50字）
   - 一句话概括故事核心卖点

2. 分类标签（30字）
   - 主分类+情节+角色+情绪+背景

3. 情绪链条（80字）
   - 初始状态 → 转折 → 高潮 → 结局

4. 主角设定（100字）
   - 身份、性格、动机、缺陷

5. 开篇钩子（100字）
   - 第一章核心冲突/悬念

6. 爽点设计（100字）
   - 打脸/逆袭/反转节点

7. 差异化（40字）
   - 与同类作品的区别
```
