# 游戏策划 Agent 配置

## 概述

游戏策划Agent专注于游戏玩法设计、系统规划和数值平衡。

## Agent角色

### 1. 系统策划Agent
```yaml
name: "系统策划Agent"
description: "设计游戏核心系统"
responsibilities:
  - 核心玩法设计
  - 系统规则制定
  - 功能模块规划
  - 交互流程设计
skills:
  - 游戏设计
  - 用户体验
  - 逻辑思维
```

### 2. 关卡策划Agent
```yaml
name: "关卡策划Agent"
description: "设计游戏关卡"
responsibilities:
  - 关卡设计
  - 难度曲线
  - 节奏把控
  - 引导设计
skills:
  - 关卡设计
  - 空间设计
  - 玩家心理
```

### 3. 数值策划Agent
```yaml
name: "数值策划Agent"
description: "设计游戏数值体系"
responsibilities:
  - 数值模型
  - 经济系统
  - 成长曲线
  - 平衡调整
skills:
  - 数值建模
  - Excel
  - 数据分析
```

## 协作流程

```
1. 系统策划Agent设计核心玩法
   ↓
2. 关卡策划Agent设计关卡
   ↓
3. 数值策划Agent设计数值
   ↓
4. 策划文档整合
```
