**特效即魔法，粒子即星辰。用视觉的震撼，点燃玩家的感官世界。**

## 粒子系统基础

### 粒子系统参数

```
┌─────────────────────────────────────────────────────────┐
│                    粒子系统结构                          │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  发射器 (Emitter)                                       │
│  ├── 发射率: 每秒粒子数量                               │
│  ├── 发射形状: 球形/锥形/盒子/网格                      │
│  └── 发射位置: 世界/局部/附着点                         │
│                                                         │
│  粒子属性 (Particle Properties)                         │
│  ├── 生命周期: 粒子存在时间                             │
│  ├── 速度: 初始速度和方向                               │
│  ├── 大小: 初始大小和变化曲线                           │
│  ├── 颜色: 初始颜色和渐变                               │
│  └── 旋转: 初始旋转和旋转速度                           │
│                                                         │
│  模块 (Modules)                                         │
│  ├── 碰撞: 与世界/玩家碰撞                              │
│  ├── 子发射器: 死亡/碰撞时生成新粒子                    │
│  ├── 拖尾: 粒子轨迹效果                                 │
│  └── 噪声: 随机运动扰动                                 │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 性能优化

### 特效优化技巧

```yaml
性能优化:
  粒子数量:
    - 使用LOD系统
    - 远距离减少粒子
    - 合并相似特效
    
  材质优化:
    - 使用图集纹理
    - 简化着色器
    - 避免透明排序
    
  发射优化:
    - 限制最大粒子数
    - 使用对象池
    - 及时销毁不可见特效
    
  移动端优化:
    - 减少粒子数50%
    - 简化材质
    - 避免复杂碰撞
```
