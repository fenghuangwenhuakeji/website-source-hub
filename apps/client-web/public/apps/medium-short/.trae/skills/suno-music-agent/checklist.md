## 快速检查流程

```
1. 需求理解 → 2. 歌词创作 → 3. 韵律检查 → 4. Suno提示词 → 5. 最终审查
   ↓              ↓              ↓              ↓              ↓
  确认主题      结构完整      押韵正确      格式正确      全部通过
  确认风格      内容质量      节奏流畅      参数合理      可以交付
  确认情绪      情感连贯      无硬押韵      兼容Suno      用户满意
```

## 常见问题排查

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 歌词不押韵 | 韵律引擎问题 | 检查韵母、调整用词 |
| Suno生成偏差 | 提示词不准确 | 优化关键词、增加参考 |
| 情感不连贯 | 情绪链设计问题 | 重新设计情绪曲线 |
| 风格不明确 | 风格词模糊 | 使用更具体的风格词 |
| 歌词不自然 | 用词生硬 | 简化表达、口语化 |
