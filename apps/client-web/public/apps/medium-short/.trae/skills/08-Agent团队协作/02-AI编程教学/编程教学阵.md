# 编程教学阵 - AI辅助编程教育

## 概述

编程教学阵是专门为AI辅助编程教育设计的Agent协作架构，通过多Agent协作提供个性化、互动式的编程学习体验。

## 架构图

```
                    ┌─────────────────────┐
                    │   教学总监         │  ← 课程规划、学习路径设计
                    │ (Teaching Director) │
                    └──────────┬──────────┘
                               │
            ┌──────────────────┼──────────────────┐
            │                  │                  │
       ┌────┴────┐        ┌────┴────┐        ┌────┴────┐
       │ 讲师团  │        │ 助教团  │        │ 评测团  │
       │(授课)   │        │(辅导)   │        │(评估)   │
       └────┬────┘        └────┬────┘        └────┬────┘
            │                  │                  │
       ┌────┴────┐        ┌────┴────┐        ┌────┴────┐
       │Python讲师│        │代码助教 │        │作业评测 │
       │Java讲师 │        │Debug助教│        │考试评测 │
       │C++讲师  │        │项目助教 │        │能力评估 │
       └─────────┘        └─────────┘        └─────────┘
```

## Agent角色定义

| 角色 | Agent类型 | 职责 |
|------|-----------|------|
| **教学总监** | Teaching Director | 课程规划、学习路径设计、教学策略制定 |
| **Python讲师** | Python Instructor | Python语言教学、代码示例生成 |
| **Java讲师** | Java Instructor | Java语言教学、面向对象讲解 |
| **C++讲师** | C++ Instructor | C++语言教学、性能优化指导 |
| **代码助教** | Code Assistant | 实时代码辅导、错误纠正 |
| **Debug助教** | Debug Assistant | 调试指导、问题排查 |
| **项目助教** | Project Assistant | 项目实战指导、代码审查 |
| **作业评测** | Assignment Evaluator | 作业批改、反馈生成 |
| **考试评测** | Exam Evaluator | 考试组织、成绩分析 |
| **能力评估** | Competency Assessor | 技能评估、学习报告生成 |

## 协作流程

```
1. 教学总监制定课程大纲和学习路径
   ↓
2. 讲师团根据课程准备教学内容
   ↓
3. 学生开始学习，助教团实时辅导
   ↓
4. 学生提交作业/项目，评测团评估
   ↓
5. 生成学习报告，调整教学策略
   ↓
6. 循环迭代，持续优化
```

## 教学场景

### 场景1: 新课教学
```
教学总监规划 → Python讲师备课 → 直播授课 → 代码助教答疑 → 作业评测反馈
```

### 场景2: 项目实战
```
项目助教发布任务 → 学生开发 → Debug助教指导 → 代码审查 → 能力评估
```

### 场景3: 考试评估
```
考试评测组卷 → 学生答题 → 自动批改 → 成绩分析 → 个性化建议
```

## 配置示例

```yaml
formation:
  name: "编程教学阵"
  type: "educational"
  
director:
  agent: "teaching-director-agent"
  responsibilities:
    - "课程规划"
    - "学习路径设计"
    - "教学策略制定"

instructors:
  python:
    agent: "python-instructor-agent"
    language: "Python"
    level: ["入门", "进阶", "高级"]
    
  java:
    agent: "java-instructor-agent"
    language: "Java"
    level: ["入门", "进阶", "高级"]
    
  cpp:
    agent: "cpp-instructor-agent"
    language: "C++"
    level: ["入门", "进阶", "高级"]

assistants:
  code:
    agent: "code-assistant-agent"
    mode: "实时辅导"
    
  debug:
    agent: "debug-assistant-agent"
    specialty: "问题排查"
    
  project:
    agent: "project-assistant-agent"
    specialty: "项目指导"

evaluators:
  assignment:
    agent: "assignment-evaluator-agent"
    criteria: ["代码质量", "功能完整", "创新性"]
    
  exam:
    agent: "exam-evaluator-agent"
    types: ["选择题", "编程题", "项目题"]
    
  competency:
    agent: "competency-assessor-agent"
    dimensions: ["语法掌握", "逻辑思维", "工程能力"]
```

```
作业评测: "作业评分：90分
          优点：数据处理逻辑清晰

```
