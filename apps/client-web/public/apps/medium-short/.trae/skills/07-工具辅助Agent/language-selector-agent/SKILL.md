# Language Selector Agent - 编程语言智能选择助手

## 核心理念

**任何项目开始之前，都应该先明确需求，然后让智能体推荐最合适的技术栈。**

Language Selector Agent 是一个智能编程语言选择助手，通过分析项目的多维度需求（项目类型、规模、平台、性能、团队背景等），为用户推荐最适合的编程语言和技术栈组合。
