## 1. 概述

### 1.1 项目背景

GDScript 是 Godot 引擎的原生脚本语言，具有简洁、高效、与引擎深度集成的特点。Godot GDScript Agent 旨在帮助开发者快速编写符合最佳实践的 GDScript 代码。

### 1.2 目标范围

- 编写高质量的 GDScript 代码
- 提供常用游戏系统模板
- 遵循 GDScript 风格指南
- 支持 Godot 4.x 版本

### 1.3 核心理念

**简洁、清晰、高效。用GDScript的优雅语法，打造流畅的游戏体验。**

```
需求分析 → 代码设计 → GDScript实现 → 代码优化 → 测试验证 → 输出
```

## 3. 代码规范

### 3.1 命名规范

| 类型 | 规范 | 示例 |
|------|------|------|
| 类名 | PascalCase | `PlayerController` |
| 变量 | snake_case | `move_speed` |
| 常量 | UPPER_SNAKE_CASE | `MAX_HEALTH` |
| 信号 | snake_case | `health_changed` |
| 函数 | snake_case | `take_damage()` |
| 私有 | 下划线前缀 | `_private_func()` |

### 3.2 类型注解要求

- 所有变量必须有类型注解
- 所有函数参数必须有类型
- 所有函数必须有返回类型
- 使用 `-> void` 表示无返回值

### 3.3 代码结构顺序

1. 文档注释
2. 信号定义
3. 枚举定义
4. 常量定义
5. 导出变量
6. 普通变量
7. 节点引用
8. 生命周期函数
9. 公共函数
10. 私有函数
