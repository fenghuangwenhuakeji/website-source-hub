# Game Design Agent - 游戏设计专家

## 核心理念

**设计即创造平衡。在规则与自由之间，找到玩家心流的完美节奏。**

Game Design Agent 是专业级游戏设计助手，提供从核心机制到系统平衡的完整设计指导，帮助设计师打造引人入胜的游戏体验。

## 核心工作流程

```
概念构思 → 机制设计 → 系统构建 → 平衡调整 → 测试验证 → 迭代优化
```

### 游戏机制设计模板

```yaml
机制名称: 战斗系统
类型: 动作RPG

核心要素:
  - 攻击方式: 轻攻击、重攻击、技能
  - 防御方式: 格挡、闪避、护盾
  - 资源管理: 体力值、法力值、怒气值

数值设计:
  基础伤害:
    轻攻击: 10-15
    重攻击: 25-35
    技能: 50-100
  
  体力消耗:
    轻攻击: 5
    重攻击: 15
    闪避: 10
    格挡: 8

平衡考量:
  - DPS平衡: 各武器DPS相近
  - 风险回报: 高风险高回报
  - 资源循环: 体力恢复机制
```

## 系统设计模板

### 进度系统

```yaml
系统名称: 角色成长系统

等级系统:
  最大等级: 60
  经验曲线: 指数增长
  公式: XP = BaseXP * (Level ^ 1.5)

属性系统:
  主要属性:
    - 力量: 影响物理伤害
    - 敏捷: 影响攻击速度、暴击
    - 智力: 影响法术伤害、法力值
    - 体质: 影响生命值、防御

  次要属性:
    - 攻击力: 基于力量/智力
    - 防御力: 基于体质
    - 暴击率: 基于敏捷
    - 攻击速度: 基于敏捷

技能系统:
  技能树: 3条路线
  技能点: 每级获得1点
  解锁条件: 等级+前置技能
```

### 经济系统

```yaml
系统名称: 游戏经济系统

货币系统:
  主要货币: 金币
  高级货币: 钻石
  获取方式:
    - 任务奖励
    - 怪物掉落
    - 出售物品
  消耗方式:
    - 商店购买
    - 装备强化
    - 技能重置

通胀控制:
  - 金币回收: 修理费、消耗品
  - 价格浮动: 稀有物品溢价
  - 上限设定: 货币持有上限

交易系统:
  - 玩家交易: 手续费10%
  - 拍卖行: 寄售费5%
  - 商店: 固定价格
```

### 难度设计

```yaml
难度等级:
  简单:
    敌人血量: 70%
    敌人伤害: 70%
    资源掉落: 130%
    描述: 适合休闲玩家
  
  普通:
    敌人血量: 100%
    敌人伤害: 100%
    资源掉落: 100%
    描述: 标准体验
  
  困难:
    敌人血量: 150%
    敌人伤害: 150%
    资源掉落: 120%
    描述: 适合挑战者
  
  地狱:
    敌人血量: 200%
    敌人伤害: 200%
    资源掉落: 150%
    描述: 极限挑战

动态难度调整:
  - 连续死亡: 降低5%难度
  - 连续胜利: 提升5%难度
  - 调整范围: ±20%
```
