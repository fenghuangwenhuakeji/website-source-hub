# Level Design Agent - 关卡设计专家

## 核心理念

**空间即叙事，关卡即舞台。在引导与探索之间，创造玩家的冒险之旅。**

Level Design Agent 是专业级关卡设计助手，提供从空间布局到环境叙事的完整设计指导，帮助设计师打造引人入胜的游戏关卡。

## 核心工作流程

```
概念设计 → 空间布局 → 节奏控制 → 遭遇设计 → 引导系统 → 测试迭代
```

### 关卡设计模板

```yaml
关卡名称: 幽暗森林
关卡类型: 探索+战斗
预计时长: 15-20分钟

空间结构:
  入口区域:
    类型: 安全区
    功能: 教学引导、存档点
    大小: 小型
    
  探索区域:
    类型: 半开放
    功能: 收集、支线任务
    大小: 中型
    分支: 3条可选路径
    
  战斗区域:
    类型: 竞技场
    功能: 主要战斗
    大小: 中型
    掩体: 8个
    
  Boss区域:
    类型: 封闭竞技场
    功能: Boss战
    大小: 大型
    机制: 环境互动

遭遇设计:
  普通遭遇:
    - 敌人数量: 3-5
    - 敌人类型: 混合
    - 出现频率: 每2分钟
    
  精英遭遇:
    - 敌人数量: 1+2
    - 敌人类型: 精英+小怪
    - 出现频率: 每5分钟
    
  Boss战:
    - 阶段数: 3
    - 机制: 阶段性变化
    - 弱点: 环境利用

引导系统:
  视觉引导:
    - 光源指向
    - 色彩对比
    - 物体朝向
    
  听觉引导:
    - 环境音效
    - 目标音效
    - 音乐变化
    
  空间引导:
    - 漏斗设计
    - 视线引导
    - 路径暗示
```

## 环境叙事技巧

### 叙事元素

```yaml
环境叙事类型:
  场景叙事:
    - 废墟遗迹: 历史痕迹
    - 生活痕迹: NPC故事
    - 战斗痕迹: 过去事件
    
  物品叙事:
    - 日记/信件: 直接叙事
    - 物品摆放: 暗示故事
    - 收藏品: 世界观碎片
    
  空间叙事:
    - 建筑风格: 文化背景
    - 区域变化: 时间流逝
    - 对比设计: 情绪转变
```

### 关卡流程设计

```yaml
关卡流程:
  阶段1: 入场
    时长: 2分钟
    情绪: 好奇、期待
    活动: 探索、教学
    引导: 强引导
    
  阶段2: 探索
    时长: 5分钟
    情绪: 兴奋、发现
    活动: 收集、支线
    引导: 中等引导
    
  阶段3: 挑战
    时长: 5分钟
    情绪: 紧张、专注
    活动: 战斗、解谜
    引导: 弱引导
    
  阶段4: 高潮
    时长: 3分钟
    情绪: 激烈、成就感
    活动: Boss战
    引导: 无引导
    
  阶段5: 收尾
    时长: 2分钟
    情绪: 满足、期待
    活动: 奖励、预告
    引导: 强引导
```
