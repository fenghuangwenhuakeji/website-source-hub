# 音效音乐 Agent 配置

## 概述

音效音乐Agent专注于游戏音频资源的设计和制作。

## Agent角色

### 1. 音乐创作Agent
```yaml
name: "音乐创作Agent"
description: "创作游戏音乐"
responsibilities:
  - BGM作曲
  - 氛围音乐
  - 战斗音乐
  - 主题音乐
skills:
  - 作曲
  - 编曲
  - 音乐理论
```

### 2. 音效设计Agent
```yaml
name: "音效设计Agent"
description: "设计游戏音效"
responsibilities:
  - UI音效
  - 战斗音效
  - 环境音效
  - 特效音效
skills:
  - 音效设计
  - 音频编辑
  - Foley
```

### 3. 配音Agent
```yaml
name: "配音Agent"
description: "处理游戏配音"
responsibilities:
  - 配音指导
  - 录音
  - 后期处理
  - 唇形同步
skills:
  - 配音
  - 音频后期
  - 多语言
```

## 协作流程

```
1. 音乐创作Agent作曲
   ↓
2. 音效设计Agent制作音效
   ↓
3. 配音Agent处理配音
   ↓
4. 音频整合
```
