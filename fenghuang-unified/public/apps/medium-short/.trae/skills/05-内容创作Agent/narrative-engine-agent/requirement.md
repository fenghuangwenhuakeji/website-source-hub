## 1. 概述

### 1.1 项目背景

短篇小说创作需要系统化的方法论支撑。Narrative Engine Agent 基于《叙事工程·元系统》，提供从构思到成文的完整创作流程。

### 1.2 目标范围

- 生成10,000-30,000字短篇小说
- 10章结构，每章1,000-3,000字
- 应用三幕式结构
- 使用零度写作技法
- 操控情绪链条

### 1.3 核心理念

**无极生太极，太极生两仪。创作始于混沌，终于心法。**

```
梗概输入 → 四象定锚 → 骨架构建 → 零度写作 → 外科精修 → 输出
```

## 3. 核心公理

### 3.1 冰山法则 (Show, Don't Tell)

**原则**:
- 不说"他很生气"，写"他捏碎了玻璃杯"
- 用动作、感官、环境传达情绪
- 删除所有"感到/觉得/似乎/仿佛"
- 每段必含2种以上感官

### 3.2 鸿沟理论 (Gap Theory)

**公式**: 故事 = 期望 ≠ 结果

**应用**:
- 每场戏必有冲突/阻碍/反转
- 主角想要A → 得到B → 引发C
- 悬念 = 信息差 + 时间压力

### 3.3 黄金螺旋 (5%-75%-15%-5%)

| 阶段 | 占比 | 功能 |
|------|------|------|
| 拉 | 5% | 建立欲望缺口 |
| 扯 | 75% | 反复拉锯升级 |
| 放 | 15% | 情绪顶点释放 |
| 收 | 5% | 余韵与升华 |

## 4. 写作技法

### 4.1 六壬：零度写作

**四总纲**:
1. **直**: 词语精准
2. **短**: 句式凝练
3. **快**: 节奏密集
4. **显**: 意图直白

**比喻清零**:
- 每1000字最多保留1个比喻
- 其余全删或改直接描述

**虚词斩杀清单**:
- 正、正在、非常、十分、极其
- 似乎、仿佛、好像、宛如、如同
- 一种、某种、那种、这种（情绪前）
- 更加、愈发、越来越、逐渐
- 开始、终于、果然

### 4.2 七星：对话法则

**规则**:
1. 每句对话必须完成：推动情节、揭示性格、制造冲突、埋设伏笔之一
2. 潜台词技术：表层 vs 深层含义
3. 方言/口癖人设化
4. 格式统一：使用「」，单句不超过30字

### 4.3 八卦：场景构建

**原则**:
- 晚进早出：直接切入冲突
- 五感强制调用：视、听、嗅、触、味
- 环境映射心情：暴雨=失控，雾天=迷茫
