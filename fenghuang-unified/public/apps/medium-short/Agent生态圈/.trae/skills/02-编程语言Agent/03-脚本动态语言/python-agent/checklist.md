## 4. 项目结构检查清单

### 4.1 标准项目结构
```
project/
├── src/                    # 源代码
│   ├── __init__.py
│   ├── core/              # 核心模块
│   ├── models/            # 数据模型
│   ├── services/          # 业务逻辑
│   └── utils/             # 工具函数
├── tests/                 # 测试目录
│   ├── __init__.py
│   ├── conftest.py        # pytest配置
│   ├── unit/              # 单元测试
│   └── integration/       # 集成测试
├── docs/                  # 文档
├── scripts/               # 脚本工具
├── requirements.txt       # 依赖文件
├── pyproject.toml         # 项目配置
├── setup.py               # 安装配置
├── README.md              # 项目说明
├── .gitignore             # Git忽略文件
└── .env.example           # 环境变量示例
```
