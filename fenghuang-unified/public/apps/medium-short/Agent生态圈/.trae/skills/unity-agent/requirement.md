## 1. 概述

### 1.1 项目背景

Unity 是全球最流行的游戏引擎之一，支持2D/3D游戏开发、AR/VR应用、跨平台部署。Unity Agent 旨在帮助开发者快速构建高质量的Unity项目。

### 1.2 目标范围

- Unity C# 脚本开发
- 2D/3D 游戏系统构建
- AR/VR 应用开发
- 跨平台部署优化
- 性能调优

### 1.3 核心理念

**一次开发，多平台部署。用C#的强大，创造无限可能的游戏世界。**

## 3. 代码规范

### 3.1 命名规范

| 类型 | 规范 | 示例 |
|------|------|------|
| 类名 | PascalCase | `PlayerController` |
| 方法 | PascalCase | `TakeDamage()` |
| 公共变量 | camelCase | `moveSpeed` |
| 私有变量 | _camelCase | `_isGrounded` |
| 常量 | PascalCase | `MaxHealth` |
| 事件 | PascalCase | `OnDeath` |

### 3.2 代码结构顺序

1. 命名空间引用
2. 类定义
3. 序列化字段
4. 公共属性
5. 私有字段
6. 生命周期方法
7. 公共方法
8. 私有方法
