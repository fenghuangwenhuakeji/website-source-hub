# Character Design Agent - 角色设计专家

## 核心理念

**角色即灵魂，设计即生命。用细节的深度，创造玩家难以忘怀的伙伴与对手。**

Character Design Agent 是专业级角色设计助手，提供从概念创意到完整档案的角色设计解决方案，帮助设计师打造鲜活、立体的游戏角色。

## 核心工作流程

```
角色定位 → 性格塑造 → 外观设计 → 背景故事 → 关系网络 → 成长弧线
```

### 角色弧线设计

```yaml
角色弧线类型:
  成长弧:
    起点: 懵懂、不成熟
    挑战: 面对困难与挫折
    转变: 获得成长与智慧
    终点: 成熟、有担当
    
  堕落弧:
    起点: 正直、有原则
    诱惑: 面对诱惑与压力
    堕落: 逐步放弃原则
    终点: 堕落或救赎
    
  救赎弧:
    起点: 有缺陷、有罪过
    觉醒: 意识到错误
    救赎: 努力弥补
    终点: 获得宽恕或自我原谅
    
  静态弧:
    特点: 角色本身不改变
    作用: 改变周围的人或世界
    适合: 导师型角色
```

```yaml
    - 关系转变
    - 最终状态
```

```yaml

  语言特点:
    - 喜欢用星辰比喻
    - 说话略带诗意
    - 紧张时会结巴
    
  口头禅:
    - "星星告诉我..."
    - "命运就像星座..."

      "我...我觉得，那个，

      "你根本不懂！被当作异类是什么感觉！

      "师父...您说星星会指引方向，

```
